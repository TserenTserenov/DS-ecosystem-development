# IWE: карта отстройки и правила языка

## Формула продукта (одна строка)

> **«IWE — среда, где интеллект и ИИ = одна система»**

Ценность: **сильный системный интеллект**. Среда — главный инструмент. Всё остальное (метод, сообщество, ЦД, подписка) — слои поддержки.

## Крючок (первый слайд / первое предложение)

> «ИИ переписывает работу. Промпт-курсы устареют вместе с моделью. Способ думать — нет.»

## Карта отстройки

| Категория | Конкретные примеры | Их ограничение | Наш ответ |
|---|---|---|---|
| **ИИ-вендоры** | Perplexity, ChatGPT-EDU, Gemini Education | Горизонтальная платформа: вы строите на их инфраструктуре, нет личного знания, нет Pack, нет домена | Вертикаль на нашей платформе: Pack + ЦД + сообщество + личная история — не забирают при смене вендора |
| **Онлайн-курсы** | Coursera, Stepik, Skillbox | Линейный контент: одинаков для всех, не адаптируется под состояние ученика, заканчивается сертификатом | Адаптивная доставка: Портной + HLR + BKT — в ответ на ваше состояние и траекторию, не расписание |
| **Инфобизнес** | Большинство TG-курсов, марафоны | Обещание результата, не инструмент. Не масштабирует экспертность — масштабирует обещание | Среда и инструменты. Результат — следствие работы в среде, не обещание продавца |
| **MBA / академия** | MBA, Executive Ed, корп. тренинги | Глубокий, но не масштабируется: диплом = конечная точка, знание не встроено в ежедневную работу | Деятельная экспертность в ежедневном потоке: знание работает внутри рабочей среды, не снаружи |
| **Коучинг / менторинг** | 1-на-1 коучи, наставники | Высокое качество, но не масштабируется физически и финансово | Экспертность переносится через среду (ЦД + Pack), не через часы человека |

**Главный аргумент отстройки (архитектурный):**
«Промпт-курсы устаревают вместе с моделью GPT-5/6. Способ думать — не устаревает никогда.»
Это не функциональная отстройка (у нас есть то, чего нет у них). Это архитектурная: их формат устареет, наш — нет.

## Правила языка

### Говорить (whitelist)

| Термин | Почему |
|--------|--------|
| **подписка** | Подчёркивает постоянную ценность, не разовую покупку |
| **среда** | Главный герой — не курс, не контент, а среда работы |
| **экосистема** | Системная взаимосвязь компонентов |
| **созидатель** | Идентичность результата — человек, создающий системы |
| **траектория** | Рост как путь, а не точка назначения |
| **мастерство** | Уровень практики, не диплом |
| **системное мышление** | Метод, а не компетенция |

### Не говорить (blacklist)

| Термин | Почему нельзя |
|--------|---------------|
| **курс** | Позиционирует нас как EdTech-платформу с линейным контентом |
| **выпускник** | Подразумевает конечную точку; у нас нет «окончания» |
| **окончание / диплом** | Тот же смысл — завершённость вместо непрерывного роста |
| **MBA-аналог** | Цепляет ненужные ассоциации (дорого, академично, не прикладное) |
| **«мы изготавливаем / мы растим вас»** | Рост — следствие работы в среде, не обещание продавца. Звучит патерналистски |
| **«научим»** | Коннотация школы, передачи знания сверху вниз |
