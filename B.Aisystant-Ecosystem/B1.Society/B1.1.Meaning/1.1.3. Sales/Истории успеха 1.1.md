# Истории успеха

## 0. Назначение документа

Этот документ описывает истории успеха в контексте раздела, посвящённого продвижение экосистемы.

---

## 1. Содержание

> Источник: транскрипция митапа «ИИ приёмы продуктивности» (Джедайская Академия, Максим Дорофеев, апр 2026). 5 участников показывали свои ИИ-инструменты продуктивности.

---

### 1.1. Даниил — системный экзокортекс (прямой пользователь IWE-подхода)

**Кто:** engineering manager в fintech-компании (IT operations + дата-центры AI на Ближнем Востоке).

**Было:** Множество задач, совещаний, доменов. Разрозненные инструменты.

**Стало:** Весь рабочий день через Claude Code с протоколами открытия/закрытия дня, паками знаний по доменам, архгейтом, MCP-интеграциями (календарь, Granola, Singularity). Готовит шаблон для передачи команде.

**Ключевая цитата:** *«Это во многом система, придуманная Цереном Цереновым из школы системного менеджмента, но с моими доработками, заточенные под меня, конкретно мой экземпляр.»*

**Что демонстрировал:**
- Протокол открытия дня (планирование, инбоксы)
- Протокол закрытия дня (настроение/энергия/фокус, синхронизация git, captures)
- Вечерняя рефлексия — *«спрашивает по всем прошедшим встречам с конца в начало, чтобы из головы выдергивать и записывать»*
- Паки знаний по доменам — аналог Pack-репо
- Архгейт — *«протокол стопа, если начинаю работать над чем-то не из плана»*
- Экстракция знаний из митингов: *«после каждого митинга экстрактор — action items в бэклог, знания в пак»*
- Хуки безопасности в Claude Code для fintech-compliance
- Передача системы команде: готовит git-репозиторий для коллег

**Реакция Максима Дорофеева:** *«Ты прям себе делаешь экзокортекс — внешнюю часть мозга.»*

**Релевантность для IWE:** Живой пользователь IWE-подхода, демонстрирующий его перед аудиторией Дорофеева. Валидация FMT-exocortex-template (готовит шаблон для команды). Fintech-сегмент (платёжеспособный).

---

### 1.2. Севара — бот «второй мозг» (не-программист)

**Кто:** предприниматель (Кыргызстан), zero coding background.

**Было:** 2 года после выгорания не могла вернуться в планирование. Пробовала менять инструменты — возвращалась на 3 дня.

**Стало:** За 5 часов (видеоурок обещал 2) создала Telegram-бота с интеграцией Obsidian + Todoist + голосовой ввод (DeepGram API). Бот автоматически классифицирует записи (задача/наблюдение/идея/бизнес-активность), попинывает за нерешённые задачи, ведёт бизнес-активности. Синхронизация 2 раза в день (14:00 и 21:00).

**Ключевая цитата:** *«Я ему научилась говорить: сделай всё сам. Вообще ничего делать не буду.»*

**Что демонстрировала:**
- Утренний бриф от бота: необработанные записи, бизнес-активности, CRM-карточки
- Голосовой ввод на ходу с автоклассификацией
- Попинывание: *«Нет прогресса с 11 марта»*
- Интеграция Obsidian → Todoist (задачи прилетают автоматически)

**Релевантность для IWE:** Валидация потребности в протоколах дня и nudge-системе. Сегмент С1 (индивидуальный практик). Доказывает: даже не-программист готов потратить 5ч на настройку «второго мозга».

---

### 1.3. Ирина — армия агентов-миньонов

**Кто:** предприниматель, активный пользователь ИИ-инструментов.

**Было:** Часы на мониторинг LinkedIn, поиск квартиры, разбор 4-летнего архива Telegram-сохранёнок.

**Стало:** Агенты в Strobery ($20/мес) автоматически мониторят LinkedIn каждые 2 часа, пишут комментарии, рассылают запросы. Квартира найдена за 36 часов (73 автоматических запроса → 20-30 откликов → 8 просмотров → 1 контракт). Архив Telegram обработан за 30 минут через Code Dispatch.

**Ключевые цитаты:**
- *«Они меня используют как кожаный мальчик — мальчишок, который капчу разгадывает.»*
- *«За 36 часов я просмотрела 30 вариантов, 8 прошла пешочком, один подошёл идеально.»*

**Релевантность для IWE:** Паттерн делегирования рутины агентам. Идея: массовая обработка inbox (Telegram-сохранёнки) — релевантно для knowledge-ingestion pipeline.

---

### 1.4. Станислав — вайб-кодер (бывший нейродиссидент)

**Кто:** «лютый нейродиссидент», считал что нейронки разжижают мозги.

**Было:** Проваливались рутинные сферы: финансы, питание, тренировки.

**Стало:** За 4 часа завайбкодил финансовое приложение (расчёт зарплаты/премий/отпускных на год с точностью до 100 руб, экспорт в ДзенМани). Меню на неделю с карточками A5. Программа тренировок из транскрипции видео эксперта. Бот в Telegram через OpenClaw на VPS.

**Ключевая цитата:** *«В первый день настройки OpenClaw ушло 100 миллионов токенов — порядка полутора тысяч евро за Sonnet.»*

**Релевантность для IWE:** Боль дороговизны кустарных решений. IWE экономит через протоколы и контекст-менеджмент. MCP-интеграции (Singularity).

---

### 1.5. Ольга — GPT-тренеры

**Кто:** использует кастомные GPT-ассистенты для личного развития.

**Было:** Внимание ускользает, отвлекается на вкладки.

**Стало:** За 1.5 месяца с GPT-тренером сосредоточенности «очень сильные изменения». Когда бросила — откат, вернулась. Также: GPT-внедрятель привычек (на основе «Атомных привычек»), GPT-коуч целей.

**Ключевая цитата:** *«Когда я хочу отвлечься, я иду к нему жаловаться вместо того, чтобы пойти почитать [ленту].»*

**Релевантность для IWE:** Тренер сосредоточенности как роль в боте. Иллюстрация различения «экзоскелет vs автопилот» — бросила → откат, значит усиливает, а не заменяет.

---

### 1.6. Михаил — геймификация задач

**Кто:** разработчик.

**Стало:** За 2-3 недели накодил мобильное приложение с героем, уровнями, энергией, рандомными наградами, паутинкой навыков.

**Релевантность для IWE:** Элементы геймификации для Activity Hub (WP-109).

## 2. Агрегированные выводы

**Главный факт:** 5 из 5 участников независимо строят кустарные «экзокортексы» из разных компонентов. Никто не доволен на 100%. IWE — единственный продукт, предлагающий это как целостную систему.

**Аудитория Дорофеева = идеальные early adopters:** уже имеют систему продуктивности, понимают ценность планирования, готовы инвестировать время и деньги ($20-1500€/мес) в настройку.

**Потенциал партнёрства:** Дорофеев планирует спринт «каждый строит себе ИИ-помощника за 12 недель». IWE может быть готовым решением для этого спринта.
