## 2. Детальное описание ИИ-ассистентов

### 2.1. Что такое ИИ-ассистент

**ИИ-Ассистент** — это диалоговый ИИ-компонент, который:
- Взаимодействует **напрямую с человеком** через чат/интерфейс
- Персонализирован под конкретную **роль человека**
- Использует данные Цифрового двойника и методику
- Помогает человеку в **развитии, обучении, рефлексии**

**Отличие от ИИ-агентов:**
- **Ассистент** — "лицо" для человека, ведет диалог
- **Агент** — "руки и ноги" ассистента, выполняет функции

Подробнее: [[Классификация ИИ-сущностей в экосистеме 3.2]]

## 3. Архитектура ИИ-ассистента

Все ИИ-ассистенты работают по единой модели **"оркестратор + агенты"**:

```
┌──────────────────────────────────────────────────────────┐
│  ИИ-АССИСТЕНТ (например, Проводник)                      │
│  ┌────────────────────────────────────────────────────┐  │
│  │  Диалоговый интерфейс (ChatGPT, Telegram, LMS)    │  │
│  └────────────────────────────────────────────────────┘  │
│                         ▼                                 │
│  ┌────────────────────────────────────────────────────┐  │
│  │  Оркестратор (роль R.GrowthGuide)                 │  │
│  │  - Понимает контекст диалога                      │  │
│  │  - Определяет нужные действия                     │  │
│  │  - Вызывает агенты через MCP                      │  │
│  └────────────────────────────────────────────────────┘  │
│                         ▼                                 │
│  Использует агенты (уровень 3б):                         │
│  - AGT.RouteGuide (формирование маршрута)                │
│  - AGT.TrajectoryPlanner (планирование траектории)       │
│  - AGT.DailyAssignmentBuilder (задания дня)              │
│  - AGT.ContentPersonalizer (персонализация)              │
│  - AGT.Evaluator (оценка артефактов)                     │
│  - AGT.RhythmKeeper (поддержка ритма)                    │
└──────────────────────────────────────────────────────────┘
```

**Ключевой принцип:**
- **Ассистент** — "лицо" для человека, ведет диалог, понимает контекст
- **Агенты** — "руки и ноги" ассистента, выполняют конкретные функции

Подробнее: [[Классификация ИИ-сущностей в экосистеме 3.2]]

## 5. Принципы разработки ассистентов

### 5.1. Персонализация

- Адаптация под конкретного человека
- Учет целей, навыков, предпочтений
- Индивидуальный темп и сложность
- Персональные примеры и формулировки

### 5.2. Прозрачность

- Объяснение каждой рекомендации
- Обоснование выбора действий
- Видимость прогресса
- Возможность корректировки маршрута

### 5.3. Эволюционность

- Непрерывное обучение на основе данных
- Адаптация к изменениям человека
- Улучшение рекомендаций со временем
- Обратная связь от пользователей

### 5.4. Интеграция

- Единый Цифровой двойник для всех ассистентов
- Общая методология и терминология
- Переход между ассистентами при смене роли
- Кооперация ассистентов при необходимости
