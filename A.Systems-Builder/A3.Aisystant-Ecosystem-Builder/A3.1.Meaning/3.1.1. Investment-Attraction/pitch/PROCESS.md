# Процесс работы над питчем

## Расположение

`DS-ecosystem-development / A.Systems-Builder / A3...Ecosystem-Builder / A3.1.Meaning / 3.1.1. Investment-Attraction / pitch/`

Семейство FX7 (Бизнесмен), ячейка 3.1 (система создания × meaning).

## Структура файлов

```
pitch/
├── PROCESS.md              # Этот файл — общий процесс (один на все версии)
├── v0.1/                   # Версия «Созидатели будущего»
│   ├── IDEAS.md            # Идеи по приоритетности (ранжированные)
│   ├── ELEVATOR.md         # 5 предложений (~30 сек)
│   ├── ONEPAGER.md         # Одностраничник A4
│   └── DECK.md             # Презентация (10 слайдов)
├── v0.2/                   # Следующая итерация
│   └── ...
└── v1.0/                   # Мажорная версия (смена нарратива)
    └── ...
```

## Принцип: Ideas-First

Все форматы питча производны от файла IDEAS. Идеи ранжированы по приоритету (Tier 1-3). Каждый формат берёт идеи сверху вниз, обрезая по объёму:

| Формат | Тиры | Объём |
|--------|------|-------|
| Elevator | Tier 1 | 5 предложений |
| Onepager | Tier 1-2 | 1 страница A4 |
| Deck | Tier 1-3 | 10 слайдов |

**Изменение идеи → обновить все форматы, где она используется.**

## Версионирование

- Каждая версия = отдельная папка (`v0.1/`, `v0.2/`, ...)
- Имя версии — в frontmatter каждого файла (`pitch_name`)
- Старые папки сохраняются (история итераций)
- Мажорная версия (`v1.0/`, `v2.0/`) = принципиальная смена нарратива
- Внутри папки файлы без суффикса версии (IDEAS.md, не IDEAS-v0.1.md)

## Когда обновлять

| Триггер | Действие |
|---------|----------|
| Новая идея / инсайт | Добавить в IDEAS → обновить форматы |
| Обратная связь от инвестора | Записать feedback → скорректировать приоритеты в IDEAS |
| Изменение продукта/метрик | Обновить Traction/прогнозы в форматах |
| Подготовка к встрече | Проверить актуальность всех форматов |

## Чеклист перед встречей

- [ ] IDEAS актуальны (приоритеты не устарели)
- [ ] Traction-цифры обновлены
- [ ] Запрос (Ask) заполнен
- [ ] Команда описана
- [ ] Elevator отрепетирован вслух (≤30 сек)
- [ ] Deck проверен визуально

## Роль (S2R)

Работа над питчем = роль **FX7 Бизнесмен / Инвестор** (система создания × предприниматель). Что видит: где растёт ценность, перспективность, распределение ресурсов. Что делает: определяет стратегию, поддерживает рост, создаёт условия для привлечения инвестиций.
