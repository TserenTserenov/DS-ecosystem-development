## Слайд 2: Проблема — знания разбросаны, AI не помогает думать

**Каждый профессионал тонет в информации, но не может организовать свои знания.**

- 12+ инструментов у среднего специалиста (Notion, Google Docs, Slack, ChatGPT, курсы...) — знания размазаны, ничего не связано
- AI отвечает на вопросы, но не знает *ваш* контекст — каждый раз с нуля
- Курсы дают контент, но не систему: 90% бросают, <15% применяют на практике
- Корпоративное обучение — разовое, без методологии, без накопления
- **Результат:** нет единого места, где знания, проекты и опыт живут, накапливаются и работают на человека

*Визуал: хаос из 12 иконок инструментов → стрелка → единая структурированная среда IWE*

## Слайд 3: Решение — персональная интеллектуальная среда

**IWE организует ваши знания, проекты и опыт. AI работает точно, потому что знает ваш контекст.**

Четыре компонента IWE:

1. **Ядро мышления** — чем думаешь. Знания организованы по принципам, а не по папкам. AI работает с *вашими* структурированными знаниями — точные ответы вместо галлюцинаций.

2. **Культура работы** — как работаешь. Протоколы, навыки и форматы — операционная дисциплина каждого дня. Не теория, а практика.

3. **Модель мастерства** — куда растёшь. 9 уровней квалификации: Ученик → Просветитель. Чем дольше в IWE — тем ценнее среда, тем точнее AI-персонализация.

4. **Сообщество созидателей** — где живёшь. Наставники, peer review, совместная практика на реальных задачах.

IWE работает поверх любой AI-платформы (Claude, ChatGPT, Copilot) — как Ubuntu работает поверх любого железа.

*Визуал: четыре слоя IWE — каждый отвечает на свой вопрос (чем? как? куда? где?)*

## Слайд 4: Ядро мышления — почему наш IWE точнее вендорского AI

**За организацией знаний стоит 30 лет формализации мышления.**

FPF (First Principles Framework) — мета-онтология: 5400+ текстов, научная база (системная инженерия, кибернетика, когнитивистика).

Пятиуровневая структура знаний:

```
ZP (Level 0)       Картина мира (физика, эволюция, математика)
    ↓
FPF (Level 0-1)    Универсальные принципы мышления (5400+ текстов)
    ↓
SPF (Level 1-2)    Прикладные принципы (инженерия, менеджмент)
    ↓
Pack (Level 2)     Предметные знания (домен участника)
    ↓
DS (Level 3)       Конкретные решения (код, планы, курсы)
```

**Что это даёт пользователю:**
- Знания организованы **по принципам** — AI работает с ними на порядок точнее
- Навык переносим: работает в инженерии, медицине, бизнесе — в любой области
- Каждый день работы обогащает среду — знания накапливаются, не теряются

**Ссылка:** [FPF](https://github.com/ailev/FPF/blob/main/Readme.md) — полная спецификация (open source)

*Визуал: пирамида ZP → FPF → SPF → Pack → DS — от картины мира к конкретным решениям*

## Слайд 5: Ваши данные — ваша собственность

**Всё в открытых форматах. Git, Markdown, YAML. Переезжайте куда хотите.**

| | Традиционный EdTech | IWE |
|---|---|---|
| **Данные** | На серверах платформы | У вас, в Git-репозитории |
| **Формат** | Проприетарный | Открытый (Markdown, YAML) |
| **При отмене подписки** | Теряете доступ к прогрессу | Забираете всё с собой |
| **AI-вендор** | Привязаны к одному | Работает с любым (Claude, GPT, Gemini) |

**Парадокс:** отсутствие lock-in — наш главный moat. Конкурент, который запирает данные, проигрывает в доверии. А тот, кто откроет данные, — не сможет дать методологию.

Switching cost растёт не через lock-in, а через ценность накопленного. Чем дольше человек с IWE — тем больше его среда знает о нём. Уйти может. Но зачем? Как уйти из Git — технически возможно, практически никто не уходит.

*Визуал: иконка «замок» (EdTech) vs «открытый ключ» (IWE)*

## Слайд 6: Продукт — что продаётся прямо сейчас

**Сегодня продаётся один продукт: подписка БР на Aisystant.**

**Одна подписка — все тиры T2–T4.** Цена растёт постепенно по мере роста ценности. Отдельных платных тиров нет.

Что получает подписчик:
- **LMS** — 9 уровней квалификации (привязка к EQF). Курсы, тесты, домашние задания с обратной связью
- **Бот-наставник** — AI в Telegram, который знает методологию и контекст участника
- **Семинары и потоки** — живая практика с наставниками (10 активных наставников)
- **ИИ как сервис** — базовый лимит AI-вызовов включён в подписку (Claude, GPT и др.). Не нужен свой API-ключ. Превышение лимита — отдельный LLM-биллинг (пакеты токенов)

**500 человек платят ежемесячно прямо сейчас. $60K ARR.**

**Путь пользователя (T0→T4, ~40 минут):**
1. /start в боте (trial 30 дней, 5 мин) → T0
2. Email (5 мин) → T1
3. Подписка БР (5 мин) → T2
4. Заполнить ЦД в боте (20 мин, включено) → T3
5. GitHub OAuth → Gateway URL (5 мин, включено) → T4 (IWE)

**Два потока дохода:** подписка БР + LLM-биллинг сверх базового лимита.

**Что строим (куда пойдёт транш):**
- **IWE-дистрибутив** — готовое окружение для интеллектуальной работы, разворачиваемое поверх AI-платформ вендоров
- **Цифровой двойник → персональное руководство.** Каждое действие записывается. Чем дольше в IWE — тем точнее руководство
- **Knowledge Gateway** — единый MCP-эндпоинт для любого AI-клиента (claude.ai, ChatGPT, Cursor, Gemini)
- **Платформенный LLM Proxy** — ИИ как сервис, роутинг по моделям, биллинг per user
- **Web App** — выход за пределы Telegram, доступ из браузера
- **Целевая цена: $40/мес** — рост ARPU через IWE + AI-персонализацию + LLM-биллинг + Web App

*Визуал: путь T0→T4 (5 шагов, 40 минут) + два столбца доходов*

## Слайд 7: Бизнес-модель — дистрибутив интеллекта

**Мы не конкурируем с AI-вендорами. Мы садимся поверх них — как Ubuntu поверх железа.**

| Аналогия | Ubuntu / Red Hat | IWE |
|---|---|---|
| **Ядро** | Linux kernel (открытый) | FPF-методология (открытый контент, 5400+ текстов) |
| **Дистрибутив** | Пакеты + конфигурация + GUI | Структура знаний + шаблоны + AI-среда |
| **За что платят** | Поддержка + Enterprise | Наставники + AI-персонализация + сообщество |
| **Vendor lock-in** | Нет (Linux = Linux) | Нет (Git + Markdown = переносимо) |

```
БЕСПЛАТНО                          ПЛАТНО
Методология, контент         →     Практика, обратная связь
Открытые форматы             →     AI-персонализация (IWE-среда)
Сообщество (базовое)         →     Наставничество, семинары
```

**Одна подписка БР** (сейчас ~$10/мес, целевая ~$40/мес). Включает все тиры T2–T4. Отдельных платных уровней нет. LLM-биллинг — отдельный поток: базовый лимит в подписке, превышение оплачивается пакетами токенов.

Наставник = ускоритель, не необходимость. Модель масштабируется без линейного роста числа наставников.

**Почему конкуренты не скопируют:** их контент = их продукт. Чтобы открыть — нужно отказаться от выручки. Мы уже открыты. Незаменяемая позиция.

*Визуал: три слоя — «Железо (AI-вендоры)» → «Дистрибутив (IWE)» → «Пользователь (его знания)»*

## Слайд 9: Moat — 7 структурных барьеров

1. **Методология (ZP + FPF)** — 30+ лет, 5400+ текстов. Не воспроизводится за деньги
2. **Цифровой двойник** — switching cost растёт с каждым месяцем. Данные = ваши, ценность среды = наша
3. **Open-content модель** — конкурент не откроет контент без потери бизнеса
4. **AI-персонализация в масштабе** — индивидуальное наставничество для тысяч
5. **Экономика вклада** — пользователи создают контент, растут в наставники. Сетевые эффекты
6. **Vendor-agnostic ОС** — работает поверх любой AI-платформы. Не конкурируем с вендорами — сажаемся на них
7. **Global core / local edge** — ядро универсально, адаптация локальная

**Чтобы скопировать, нужно 5–10 лет и отказ от текущей бизнес-модели.**

## Слайд 11: Vision — среда на всю жизнь

**Образование заканчивается. Развитие — нет.**

Вуз даёт диплом за 4 года. Курс — сертификат за 3 месяца. IWE — среда, которая растёт вместе с человеком на протяжении всей карьеры и жизни.

**Почему это не просто слоган:**
- Цифровой двойник накапливает опыт годами — чем дольше, тем ценнее
- Модель мастерства (9 уровней) — лестница на десятилетия, не разовый курс
- Родительский сценарий: родитель подключает ребёнка → ребёнок растёт вместе со своим IWE → непрерывное развитие вместо дискретных курсов

**Экономика на всю жизнь:**

| | Стоимость | Период | Что остаётся после |
|---|---|---|---|
| **Высшее образование** | $100 000–250 000 (США) | 4 года | Диплом |
| **Корпоративное L&D** | $1 500/год на сотрудника | Пока работаешь | Ничего |
| **IWE** | $120–480/год | Вся жизнь | Все знания, проекты, опыт — в ваших руках |

**Это не замена образования. Это следующая эволюция: от дискретных курсов к непрерывной среде развития.**
