# Идеи питча v0.1 «Созидатели будущего»

> Этот файл — основа для всех форматов питча. Идеи ранжированы по приоритету: от ядра к периферии. Каждый формат (5 предложений, A4, презентация) использует идеи сверху вниз, обрезая по объёму.

---

## Tier 1: Ядро (в любом формате)

### 1. Проблема: мышление под угрозой

ИИ генерирует контент за секунды. Образование продаёт доступ к контенту. Контент стал commodity — и перестал быть конкурентным преимуществом. Completion rate MOOC: 5-10%. Перенос в практику: <15%. Люди не учатся — они покупают иллюзию обучения.

**Глубже:** Цивилизационный кризис образования. Старые институты (университеты, MBA, курсы) не масштабируют персонализацию. Новые (ChatGPT-обёртки) не дают структуры и проверки. Между ними — пустая ниша.

### 2. Аудитория: созидатели с видением

Не все люди. Созидатели — те, у кого есть видение будущего и намерение его реализовать. Предприниматели, инженеры, исследователи, руководители. Те, кому нужно не «освоить навык», а **перестроить мышление** для работы со сложностью.

**ICP (текущий):** профессионалы 25-45, доход выше среднего, русскоязычные. Триггер: «хочу думать системно». 500 платных = доказанный спрос.

**Три сегмента:**
- B2C ядро: индивидуальные подписчики (€10-100/мес)
- B2B: компании, обучающие сотрудников (10 заказчиков)
- Наставники: мастера из числа выпускников (масштабируемый supply)

### 3. Решение: экосистема развития, а не EdTech

Мы не ещё один EdTech. Мы создаём **экосистему развития** — среду, в которой человек развивает системное мышление, применяет его в реальных проектах и растёт вместе с сообществом практиков.

Три компонента:
- **Системное мышление** (методология FPF) — операционная система мышления, не набор курсов
- **IWE** (Intellectual Work Environment) — персональная среда интеллектуальной работы с ИИ
- **Сообщество** — peer review, наставничество, совместные проекты

### 4. Защита мышления в эпоху ИИ

ИИ усиливает мышление, а не заменяет его (принцип экзоскелета). IWE — среда, где человек работает с ИИ как с инструментом, сохраняя авторство решений. Не «ИИ думает за вас», а «вы думаете лучше с ИИ».

## Tier 2: Усиление (A4 + презентация)

### 5. Бизнес-модель: открытый контент + платная практика

Модель Red Hat: контент бесплатен, инфраструктура практики платная. Конкуренты не могут скопировать — их контент = продукт; открыть = потерять выручку.

- T2 Изучение: $10-30/мес
- T3 Персонализация: $30-100/мес
- T4 IWE: $100+/мес
- LTV/CAC: 20-50x

### 6. Семь структурных барьеров

1. Методология (FPF) — 10+ лет разработки, невоспроизводима
2. Цифровой двойник — switching costs, 5 проекций
3. Open-content модель — конкурент не может открыть свой контент
4. ИИ-персонализация по цене MOOC — качество коучинга, цена курса
5. Экономика вклада — баллы, peer review, сетевые эффекты
6. Multi-surface — Web, Telegram, CLI, Discord
7. Global Core + Local Edge — масштабирование без редизайна

### 7. Маховики роста

- Контент → пользователи → вклад → контент (улучшение)
- Данные ЦД → ИИ точнее → retention → больше данных
- Наставники → качество → retention → выручка → наставники

### 8. Ценность по ролям

| Роль | Что получает | Измеримый эффект |
|------|-------------|-----------------|
| Ученик | Персональный маршрут | 10+ часов развития/нед |
| Интеллектуал | Мыслительное мастерство | +20-50% к доходу |
| Профессионал | Прикладное мастерство | Premium-ставки |
| Исследователь | Исследовательское мастерство | Публикации, гранты |
| Просветитель | Платформенный доход | Пассивный доход |

### 14. Конкурентный ландшафт

| Альтернатива | Что продаёт | Фатальная слабость |
|---|---|---|
| **Coursera, Skillbox** | Доступ к контенту | Нет практики, retention <20% |
| **MBA / университеты** | Сигнал статуса | Дорого ($100K+), негибко |
| **Коучинг** | Часы эксперта ($300/час) | Не масштабируется |
| **ChatGPT + обёртки** | Инструмент | Нет структуры, нет проверки |
| **Корп. обучение** | Compliance | Не для человека, для отчётности |

**Наша ниша:** персонализация коучинга по цене MOOC. Ни один конкурент не может занять этот квадрант — для этого нужно одновременно иметь методологию + ИИ + открытый контент + цифровой двойник.
