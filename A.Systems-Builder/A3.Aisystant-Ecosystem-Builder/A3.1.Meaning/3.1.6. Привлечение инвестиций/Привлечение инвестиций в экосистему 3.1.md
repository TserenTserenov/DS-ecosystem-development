# Привлечение инвестиций в экосистему

## 0. Назначение документа

Этот документ описывает **стратегию и набор документов**, необходимых для привлечения инвестиций в экосистему развития. Служит руководством для подготовки к раундам финансирования и взаимодействия с инвесторами.

**Адресаты:**
- Основатели и команда фандрайзинга
- Финансовый директор (CFO)
- Юридическая команда
- Консультанты по привлечению инвестиций

**Что даёт этот документ:**
- Полный чек-лист документов для инвесторов
- Структура и содержание каждого документа
- Процесс взаимодействия с инвесторами
- Типы инвесторов и их особенности

## 2. Питч-документы

### 2.1 Elevator Pitch (30 секунд)

**Структура:**
```
[Проблема] + [Решение] + [Уникальность] + [Запрос]
```

**Шаблон:**

> «Мир стал сложнее, чем способность людей его осмыслять. [НАЗВАНИЕ] — это экосистема для созидателей, где участники получают персональную траекторию развития через ИИ-проводника, зарабатывают токены за реальный вклад и применяют системное мышление к своим проектам. Мы уже [TRACTION]. Поднимаем [СУММА] на [ЦЕЛИ]».

**Требования к Elevator Pitch:**
- [ ] Не более 100 слов
- [ ] Понятен человеку вне индустрии
- [ ] Содержит числовой traction
- [ ] Чёткий call-to-action

## 3. Pitch Deck (Презентация для инвесторов)

### 3.1 Структура презентации (12-15 слайдов)

| # | Слайд | Содержание | Время |
|---|-------|------------|-------|
| 1 | Титульный | Название, логотип, tagline, контакты | 10 сек |
| 2 | Проблема | Боль рынка с данными и примерами | 60 сек |
| 3 | Решение | Как решаем проблему (высокий уровень) | 60 сек |
| 4 | Продукт | Демо / скриншоты / архитектура | 90 сек |
| 5 | Как это работает | User journey или процесс | 60 сек |
| 6 | Рынок | TAM/SAM/SOM с источниками | 45 сек |
| 7 | Бизнес-модель | Как зарабатываем, unit economics | 60 сек |
| 8 | Traction | Метрики, графики роста, кейсы | 60 сек |
| 9 | Конкуренты | Карта позиционирования | 45 сек |
| 10 | Go-to-market | Стратегия роста и каналы | 45 сек |
| 11 | Команда | Фото, опыт, почему именно мы | 60 сек |
| 12 | Финансы | Прогноз выручки, burn rate | 45 сек |
| 13 | Запрос | Сумма, условия, использование | 30 сек |
| 14 | Roadmap | Milestones на 12-24 месяца | 30 сек |
| 15 | Контакты | Как связаться, следующие шаги | 10 сек |

### 3.2 Детальное содержание ключевых слайдов

#### Слайд «Проблема»

**Содержание для экосистемы:**
- Мир стал сложнее, чем способность людей его осмыслять
- Традиционное образование: дипломы вместо траекторий
- $X трлн теряется на неэффективном обучении
- Статистика: X% сотрудников не применяют знания после курсов

#### Слайд «Решение»

**Содержание для экосистемы:**
- Экосистема созидателей: учёба + работа + сообщество + признание
- Две ключевые фишки:
  1. Экономика вклада через токены
  2. Персональная траектория с цифровым двойником
- Визуальная схема экосистемы

#### Слайд «Продукт»

**Содержание для экосистемы:**
- ИИ-Проводник: персональный маршрут развития
- Цифровой двойник: история и траектория участника
- Платформа обучения с интерактивными модулями
- Система токенов за вклад
- Скриншоты интерфейса

#### Слайд «Рынок»

**Расчёт TAM/SAM/SOM:**

| Сегмент | Размер | Расчёт |
|---------|--------|--------|
| **TAM** | $X млрд | Глобальный рынок EdTech + Corporate Learning |
| **SAM** | $Y млрд | Сегмент lifelong learning + AI-enhanced |
| **SOM** | $Z млн | Достижимый рынок за 5 лет |

#### Слайд «Бизнес-модель»

**Источники дохода экосистемы:**

| Поток | Описание | Доля |
|-------|----------|------|
| Подписки B2C | Стажировки и резидентуры | X% |
| Корпоративные программы B2B | Резидентуры для команд | Y% |
| ИИ-агенты (SaaS) | Лицензии на агентов | Z% |
| Токеномика | Комиссии с транзакций | W% |

#### Слайд «Traction»

**Метрики для показа:**
- График роста пользователей (MAU)
- График роста выручки (MRR)
- Retention по когортам
- Ключевые партнёрства
- Цитаты пользователей

#### Слайд «Конкуренты»

**Карта позиционирования:**

```
              Персонализация
                    ↑
                    │
    Coursera        │        [МЫ]
    Udemy           │
    ────────────────┼────────────────→ Сообщество
                    │                  и практика
    Традиционные    │        Cohort-based
    университеты    │        courses
                    │
```

**Конкурентные преимущества:**
1. Цифровой двойник и персональная траектория
2. Токеномика и экономика вклада
3. Системное мышление как методология
4. Экосистема для жизни, не только обучения

#### Слайд «Команда»

**Формат:**
```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  [ФОТО]  Имя Фамилия, CEO                                       │
│          15+ лет в EdTech, ex-[Компания]                        │
│          [Достижение]                                           │
│                                                                 │
│  [ФОТО]  Имя Фамилия, CTO                                       │
│          ML/AI эксперт, ex-[Компания]                           │
│          [Достижение]                                           │
│                                                                 │
│  [ФОТО]  Имя Фамилия, CPO                                       │
│          Product в [Компании]                                   │
│          [Достижение]                                           │
│                                                                 │
│  Advisors: [Имена известных людей]                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### Слайд «Запрос»

**Содержание:**

| Параметр | Значение |
|----------|----------|
| Размер раунда | $X млн |
| Инструмент | SAFE / Convertible Note / Equity |
| Оценка (если применимо) | $Y млн pre-money |
| Минимальный чек | $Z тыс |

**Использование средств:**
- 50% — Продукт и разработка
- 25% — Маркетинг и рост
- 15% — Команда
- 10% — Операции

| Метрика | Формула | Целевое значение |

## 6. Типы инвесторов и подходы

### 6.1 Типы инвесторов

| Тип | Чек | Особенности | Подход |
|-----|-----|-------------|--------|
| **Business Angels** | $25K-$500K | Быстрые решения, личные отношения | Warm intro, networking |
| **Micro VCs** | $100K-$2M | Фокус на ранних стадиях | Pitch events, accelerators |
| **Seed Funds** | $500K-$5M | Профессиональный DD | Structured process |
| **Series A VCs** | $5M-$20M | Требуют product-market fit | Metrics-driven |
| **Corporate VCs** | $1M-$20M | Стратегический интерес | Business development angle |
| **Impact Investors** | $500K-$10M | Фокус на social impact | Impact metrics, theory of change |
| **Family Offices** | $500K-$10M | Долгосрочный горизонт | Personal relationships |

### 6.2 Целевые инвесторы для экосистемы

**Профиль идеального инвестора:**
- Опыт в EdTech / AI / Consumer Social
- Понимание token economics
- Горизонт 7-10 лет
- Готовность к регуляторным рискам (токены)
- Value-add: нетворк, экспертиза, операционная помощь

**Критерии приоритизации:**

| Критерий | Вес |
|----------|-----|
| Релевантный портфель | 25% |
| Value-add потенциал | 25% |
| Размер чека / stage fit | 20% |
| Скорость принятия решений | 15% |
| Условия (terms) | 15% |

### 6.3 Стратегия взаимодействия по типам

**Business Angels:**
```
Подход: Личные отношения, storytelling
Timeline: 2-4 недели
Документы: One-pager, короткий deck
Follow-up: Регулярные апдейты, вовлечение в продукт
```

**Venture Funds:**
```
Подход: Структурированный процесс
Timeline: 2-4 месяца
Документы: Полный deck, финмодель, data room
Follow-up: Еженедельные статус-апдейты
```

**Strategic / Corporate:**
```
Подход: Business development первым
Timeline: 3-6 месяцев
Документы: Partnership proposal, затем investment
Follow-up: Совместные проекты, пилоты
```

## 8. Term Sheet и условия

### 8.1 Ключевые параметры Term Sheet

| Параметр | Описание | На что смотреть |
|----------|----------|-----------------|
| **Valuation** | Pre-money оценка | Адекватность рынку и метрикам |
| **Investment Amount** | Размер инвестиции | Достаточность для milestones |
| **Instrument** | SAFE / Convertible / Equity | Простота vs. защита |
| **Discount** | Скидка при конвертации | Стандарт 15-25% |
| **Valuation Cap** | Потолок оценки при конвертации | Не слишком низкий |
| **Pro-rata Rights** | Право участия в следующих раундах | Стандартно для лидов |
| **Board Seats** | Места в совете директоров | Не больше 1 на seed |
| **Liquidation Preference** | Приоритет при ликвидации | 1x non-participating стандарт |
| **Anti-dilution** | Защита от размытия | Broad-based weighted average |
| **Vesting** | Vesting основателей | 4 года с 1 год cliff |

### 8.2 Красные флаги в Term Sheet

**Избегать:**
- [ ] Liquidation preference >1x
- [ ] Full participating preferred
- [ ] Full ratchet anti-dilution
- [ ] Excessive board control
- [ ] Pay-to-play provisions (на ранних стадиях)
- [ ] Redemption rights
- [ ] Cumulative dividends

### 8.3 Шаблон SAFE (для ранних стадий)

**Параметры Y Combinator SAFE:**
- Valuation Cap: $X
- Discount: Y% (опционально)
- Pro-rata: Да/Нет
- MFN: Да/Нет

### 9.1 Investor Update (ежемесячный)

**Структура:**

```
Subject: [Company Name] — Monthly Update [Month Year]

   - Ключевое достижение

   - Главный вызов

2. Метрики
   | Метрика | Прошлый месяц | Этот месяц | Изменение |
   |---------|---------------|------------|-----------|
   | MRR     | $X            | $Y         | +Z%       |
   | Users   | X             | Y          | +Z%       |
   | ...     | ...           | ...        | ...       |

3. Highlights
   - Достижение 1
   - Достижение 2
   - Достижение 3

4. Challenges
   - Проблема 1 и как решаем
   - Проблема 2 и как решаем

5. Asks
   - Нужна помощь с X
   - Ищем intro к Y

6. Roadmap / Next Month
   - Цель 1
   - Цель 2
   - Цель 3
```

### 9.2 Правила коммуникации

**Частота:**
- Monthly updates — всем инвесторам
- Quarterly calls — с лидами раунда
- Ad-hoc — при важных событиях

**Принципы:**
1. **Прозрачность** — плохие новости сообщать первыми
2. **Консистентность** — регулярный ритм апдейтов
3. **Конкретность** — числа и факты, не общие слова
4. **Asks** — всегда просить о конкретной помощи

## 10. Специфика привлечения для экосистемы

### 10.1 Уникальные аспекты pitch'а

**Что подчеркивать:**

1. **Экономика вклада:**
   - Токеномика как инструмент engagement и retention
   - Снижение CAC через viral loops
   - Дополнительный revenue stream

2. **Цифровой двойник:**
   - Data moat — накопление данных о траекториях
   - Персонализация как конкурентное преимущество
   - Переносимость и lifetime value

3. **Network effects:**
   - Больше участников → больше контента → больше ценности
   - Peer-to-peer learning и менторство
   - Marketplace динамика

4. **Impact story:**
   - Цивилизационная амбиция
   - Демократизация образования
   - Культурная трансформация

### 10.2 Риски и митигации

| Риск | Митигация | Как представлять |
|------|-----------|------------------|
| Регуляторные риски токена | Utility token, не security; юридическая экспертиза | Показать legal opinion |
| Зависимость от ИИ-провайдеров | Multi-provider стратегия; собственные модели | Архитектурная гибкость |
| Конкуренция с EdTech гигантами | Нишевая дифференциация; community moat | Позиционирование |
| Сложность product-market fit | Итеративная валидация; метрики engagement | Текущий traction |

### 10.3 Метрики для EdTech/Community инвесторов

**Engagement метрики:**
- DAU/MAU ratio
- Time spent per session
- Content creation rate
- Peer interaction rate

**Learning outcomes:**
- Completion rate
- Transfer to practice rate
- Certification rate
- NPS

**Community health:**
- Active contributors %
- Mentor-to-mentee ratio
- Referral rate
- Churn by cohort
