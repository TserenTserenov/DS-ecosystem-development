# Practicum-Messages — Сообщения для потоков практикумов

> **Семейство:** F8 (Operations × SoI)
> **Ядро:** A (Созидатель)
> **Назначение:** Операционный контент для проведения практикумов

## Файлы

| Файл | Описание |
|------|----------|
| `ss-messages.md` | Сообщения для практикума «Системное саморазвитие» (22 шт) |
| `ms-messages.md` | Сообщения для практикума «Методы саморазвития» (22 шт) |
| `upload-to-lms.py` | Скрипт загрузки в LMS Aisystant |

## Workflow

1. Редактируете `ss-messages.md` / `ms-messages.md` — это source-of-truth
2. Запускаете `upload-to-lms.py` — загружает в LMS как новый шаблон
3. Удаляете старый шаблон через интерфейс LMS
4. Привязываете новый шаблон к потоку

```bash
# Проверить без загрузки
export AISYSTANT_SESSION_TOKEN='your-token'
python3 upload-to-lms.py --dry-run

# Загрузить
python3 upload-to-lms.py

# Посмотреть шаблоны в LMS
python3 upload-to-lms.py --list
```

## Структура сообщений

- **День -3:** Приветствие (за 3 дня до установочной встречи)
- **День 1:** Неделя 1, основное сообщение (понедельник)
- **День 2:** Неделя 1, интеллектуальные партнёры (вторник)
- **День 5:** Неделя 1, перед встречей (пятница)
- **День 8, 12:** Неделя 2 (понедельник, пятница)
- ...каждые 7 дней...
- **День 64, 68:** Неделя 10 — подготовка к защите
