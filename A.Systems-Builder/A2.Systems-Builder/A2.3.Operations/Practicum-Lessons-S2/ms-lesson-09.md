<details open>
<summary><b>Блок 2. Планирование — обсуждение (10 мин)</b></summary>

**Цель:** Показать, что планирование — это мост между стратегией и ежедневными действиями.

**Ключевая идея**

**Планирование переводит стратегию в конкретные работы.** На прошлой неделе мы ответили на «куда и зачем» (стратегия). Сегодня отвечаем на «как и когда» (план).

**Три уровня планирования**

| Уровень | Горизонт | Вопрос | Артефакт |
|---------|----------|--------|----------|
| **Стратегия** | Месяцы-годы | Куда и зачем? | Список НЭП, цели, проекты |
| **План недели** | 7 дней | Что делаю на этой неделе? | WeekPlan (список РП + бюджеты) |
| **План дня** | 1 день | Что делаю сегодня? | DayPlan (2-4 РП на день) |

**Каскад артефактов (v10) — 8 фаз**

В занятии 8 каскад стратегирования и планирования показан **по ритму** (v9, 7 стадий). Те же практики, рассмотренные **по артефактам** (строгая цепочка «выход фазы N = вход фазы N+1»), дают 8 фаз. Это представление удобно именно для планирования: видно, какой артефакт откуда приходит.

| Ф | Активность | Вход | Действие | Выход |
|---|-----------|------|----------|-------|
| **Ф1** | Восприятие сигнала | Информация извне | Обратить внимание; записать мимолётную заметку | Мимолётная заметка |
| **Ф2** | Разбор и обновление НЭП | Мимолётная заметка + список НЭП | Классифицировать; триада «проблема+неудовлетворённость+эмоция»; назначить System Level | Обновлённый список НЭП |
| **Ф3** | Обновление стратегии | Обновлённый список НЭП + стратегия + проекты + закрытый план прошлой недели | Пересмотреть гипотезы; оценить состояние; ТОС месяца; R1-RN; калибр | Обновлённая стратегия |
| **Ф4** | Формирование плана недели | Обновлённая стратегия + реестр РП + бюджет + календарь | Отобрать 1-7 РП; обновить реестр; time-boxing; ТОС недели; зафиксировать | **План недели** |
| **Ф5** | Открытие дня | План недели + вчерашний Day.md | Inbox Triage; отобрать 3-5 РП; бюджет; календарь | **План дня** |
| **Ф6** | Исполнение | План дня + контексты РП | Работа; коммиты; capture-to-pack | Артефакты РП |
| **Ф7** | Закрытие дня | Артефакты РП + план дня | Факт vs план; 3 варианта; KE-маршрутизация | Закрытый план дня |
| **Ф8** | Закрытие недели и месяца | Для недели: 7 закрытых планов дня + план недели + учёт + НЭП. Для месяца дополнительно: 4 закрытых недели + стратегия + проекты | Мультипликатор; стоп-лист; итоги; сверить РП↔НЭП; для месяца — ретро, состояние, ревизия проектов, новый фокус, калибр | Закрытый план + обновлённый список НЭП; при месяце — новая стратегия |

**Ключевая мысль:** план недели (Ф4) и план дня (Ф5) — не произвольные списки, а **выход** из цепочки артефактов. Если в плане недели РП не связан с НЭП → стратегией → проектом, цепочка сломана на одной из фаз Ф2-Ф3. Диагностика планирования начинается с проверки цепочки.

**Обсуждение**

1. Планируете ли вы свою неделю? Как именно?
2. Что происходит, когда неделя не спланирована?
3. Сколько времени тратите на планирование — и сколько это экономит?

<summary><b>Блок 3. Работа vs задача — различение + РП (10 мин)</b></summary>

**Цель:** Закрепить различение «работа ≠ задача» из занятия 2 и показать, как РП используется в планировании.

**Напоминание различения (из занятия 2)**

| | Задача | Работа (= РП) |
|---|--------|----------------|
| **Формулировка** | «Разобраться с...», «Поработать над...» | «Черновик статьи — 800 слов», «3 заметки по главе» |
| **Результат** | Непонятно, что получится | Конкретный артефакт |
| **Критерий готовности** | «Когда устану» | «Когда 800 слов написаны» |
| **Бюджет** | «Сколько получится» | «2 помидорки (50 мин)» |

**РП в плане недели**

Каждая строчка плана — это **рабочий продукт**, а не задача:

| ❌ Задачи (так не надо) | ✅ РП (так надо) |
|-------------------------|-----------------|
| Позаниматься саморазвитием | 3 образовательные заметки по разделу «Стратегирование» |
| Поработать над статьёй | Черновик поста для Telegram — 500 слов |
| Подумать о целях | Заполненная цепочка НЭП → цели → методы → проекты |
| Сходить на занятие | Конспект занятия + 2 инсайта |

**Ключевая мысль:** План из задач — это список намерений. План из РП — это список результатов. Разница огромна: РП можно проверить, показать, оценить.

<details open>
<summary><b>Блок 4. План недели: структура (РП + бюджет + роль) (15 мин)</b></summary>

**Цель:** Показать структуру плана недели и дать попрактиковаться.

**Структура плана недели**

Каждый РП в плане содержит три элемента:

| Элемент | Что это | Пример |
|---------|---------|--------|
| **РП** | Конкретный артефакт с критерием готовности | «Черновик поста — 500 слов, опубликован в клубе» |
| **Бюджет** | Оценка времени в помидорках или часах | 2ч (4 помидорки) |
| **Роль** | В какой роли создаёте этот РП | Ученик / профессионал / наставник |

**Пример плана недели**

| # | РП | Бюджет | Роль |
|---|-----|--------|------|
| 1 | 3 образовательные заметки по разделу «Планирование» | 1.5ч | Ученик |
| 2 | Черновик поста про стратегическую сессию — 500 слов | 2ч | Ученик |
| 3 | Аудит подписок: удалить 10 каналов, добавить 3 полезных | 0.5ч | Ученик |
| 4 | Стратегическая сессия: обновить НЭП и проекты | 1.5ч | Ученик |
| 5 | Подготовка к защите: собрать артефакты за 9 недель | 1ч | Ученик |
| | **Итого** | **6.5ч** | |

**Бюджет времени**

**Правило:** Суммарный бюджет РП на неделю должен быть реалистичным. Для участников практикума: **4-8 часов в неделю** на саморазвитие — реальный диапазон.

**Ловушка:** Запланировать 20 часов и не выполнить ничего. Лучше 5 РП по 1 часу, чем 1 РП на 10 часов.

**Упражнение (7 мин)**

Каждый участник составляет план на следующую неделю:
- 3-5 РП с бюджетами и ролями
- Суммарный бюджет ≤ 8 часов
- Каждый пункт — артефакт, не задача

<summary><b>Блок 5. Интеграция 8 практик: как они работают вместе (10 мин)</b></summary>

**Цель:** Показать, что 8 практик — не изолированные навыки, а система. Конвейер + ОРЗ + тройка объектов внимания.

**Три связующих каркаса**

**1. Творческий конвейер (ВДВ):** Потребление → Разбор → Размышление → Публикация

| Стадия конвейера | Какие практики работают |
|------------------|------------------------|
| Потребление | Медленное чтение (практика 2) |
| Разбор | Мышление письмом (практика 3) |
| Размышление | Мышление проговариванием (практика 4) |
| Публикация | Мышление письмом + проговаривание (практики 3, 4) |

**2. ОРЗ (Открытие → Работа → Закрытие):** фрактальный паттерн на трёх масштабах

| Масштаб | Открытие | Работа | Закрытие |
|---------|----------|--------|----------|
| Сессия | Стоп-момент + РП | Помидорка | Что сделал, что на потом |
| День | DayPlan | Работа по плану | Похвала + передача контекста |
| Неделя | WeekPlan (стратегическая сессия) | Работа по плану | WeekReview |

**3. Три объекта внимания:** каждая практика усиливает один или несколько объектов

| Практика | Мировоззрение и цель | Навыки и инструменты | Ресурсы и окружение |
|----------|---------------------|---------------------|---------------------|
| 0. О практиках и втором мозге | ● | ● | |
| 1. Инвестирование и учёт времени | | ● | ● |
| 2. Медленное чтение | ● | ● | |
| 3. Мышление письмом | | ● | |
| 4. Мышление проговариванием | | ● | |
| 5. Организация досуга | | | ● |
| 6. Формирование окружения | | | ● |
| 7. Стратегирование | ● | | |
| 8. Планирование | ● | ● | ● |

**Ключевая мысль:** Планирование (практика 8) — это интегратор. В плане недели видны все практики: какие работают, какие выпали. Если в плане нет ни одного РП на чтение — значит, конвейер остановился на входе.

</details>

<summary><b>Блок 6. Подготовка к защите: чек-лист мастерства + сбор артефактов (15 мин)</b></summary>

**Цель:** Подготовить участников к финальному занятию — защите. Дать структуру рассказа и критерии оценки.

**Что такое защита**

На занятии 10 каждый участник расскажет о своём пути за 10 недель: что делал, чему научился, что изменилось. Формат: **рассказ 5-7 минут** + вопросы группы.

**Чек-лист мастерства (8 практик × 4 степени)**

Заполните таблицу — честная самооценка по каждой практике:

| # | Практика | Объяснение | Умение | Навык | Мастерство |
|---|----------|------------|--------|-------|------------|
| 0 | О практиках и втором мозге | ☐ | ☐ | ☐ | ☐ |
| 1 | Инвестирование и учёт времени | ☐ | ☐ | ☐ | ☐ |
| 2 | Медленное чтение | ☐ | ☐ | ☐ | ☐ |
| 3 | Мышление письмом | ☐ | ☐ | ☐ | ☐ |
| 4 | Мышление проговариванием | ☐ | ☐ | ☐ | ☐ |
| 5 | Организация досуга | ☐ | ☐ | ☐ | ☐ |
| 6 | Формирование окружения | ☐ | ☐ | ☐ | ☐ |
| 7 | Стратегирование | ☐ | ☐ | ☐ | ☐ |
| 8 | Планирование | ☐ | ☐ | ☐ | ☐ |

**Напоминание степеней:**
- **Объяснение** — понимаю, зачем и как
- **Умение** — делаю регулярно, но с усилием (цель практикума)
- **Навык** — делаю автоматически
- **Мастерство** — творчески адаптирую

**Сбор артефактов**

Для защиты подготовьте **портфолио** — конкретные артефакты за 9 недель:

| Неделя | Практика | Артефакт |
|--------|----------|----------|
| 1 | О практиках | Карта конвейера, самооценка по 8 практикам |
| 2 | Учёт времени | Таблица учёта за неделю |
| 3 | Медленное чтение | 3 образовательные заметки |
| 4 | Мышление письмом | Черновик (заготовка в клубе) |
| 5 | Проговаривание | Аудиозапись или конспект проговаривания |
| 6 | Досуг | Карта досуга |
| 7 | Окружение | Аудит окружения (3 среды) |
| 8 | Стратегирование | Цепочка НЭП → цели → методы → проекты |
| 9 | Планирование | План недели (РП + бюджеты) |

**Структура рассказа на защите (5-7 мин)**

1. **Откуда начал:** На какой ступени ученика был? Какие практики были на нуле?
2. **Что делал:** Какие практики освоил? Какие артефакты создал? (показать 2-3 лучших)
3. **Что изменилось:** Какие привычки появились? Что стало частью ритма?
4. **Куда дальше:** Какие практики продолжу? Какая стратегия на следующий месяц?

<summary><b>Блок 7. Ступени ученика: финальная самооценка (10 мин)</b></summary>

**Цель:** Оценить свой прогресс за 9 недель — с какой ступени начали, на какой сейчас.

**Напоминание ступеней (из занятия 1)**

| Ступень | Характеристика | Триггер | Ритм |
|---------|---------------|---------|------|
| **1. Случайный** | Учусь когда заставили | Внешнее давление | ≤2 ч/нед |
| **2. Практикующий** | Учусь по настроению | Стимул (курс, дедлайн) | ≥2 слота/нед |
| **3. Систематический** | Учусь почти каждый день | Привычка | Ежедневно, ≥6 ч/нед |
| **4. Дисциплинированный** | Строю личную траекторию | Внутренняя цель | ≥10 ч/нед |
| **5. Проактивный** | Строю траекторию + передаю культуру | Миссия | ≥10 ч/нед устойчиво |

**Самооценка**

Каждый участник отвечает:
1. **На какой ступени я начал практикум?** (неделя 1)
2. **На какой ступени я сейчас?** (неделя 9)
3. **Что позволило перейти?** (конкретная практика, привычка, изменение)
4. **Что мешает перейти на следующую ступень?**

**Обсуждение**

- У кого произошёл переход между ступенями? Что стало триггером?
- Кто остался на той же ступени? Что помешало?
- Цель программы — проактивный ученик с IWE. Что ещё нужно для этого?

</details>

<summary><b>Блок 8. Задание ИП (итоговое) + рефлексия (10 мин)</b></summary>

**Цель:** Выдать итоговое задание для ИП, собрать обратную связь.

**Задание на неделю (из сообщения в боте, день 57)**

- Три фокуса: работа ≠ задача, бюджет времени, план недели
- Практика: составить план недели с РП и оценкой времени
- Подготовить портфолио артефактов для защиты

**Задание для интеллектуальных партнёров — итоговое (из сообщения в боте, день 61)**

Каждый составляет план на неделю по правилам (РП + бюджет + роль). Партнёр проверяет:

1. **Каждая работа с продуктом?** Нет задач без артефакта?
2. **Бюджет реалистичен?** Суммарно ≤ 8 часов? Не раздуты оценки?
3. **Все 8 практик учтены — или какие-то выпали?** Если выпали — почему?
4. **Какие практики стали привычкой?** Какие требуют усилия?

Дополнительно: помогите друг другу подготовить рассказ для защиты. Проговорите по структуре (откуда начал → что делал → что изменилось → куда дальше). Партнёр даёт обратную связь: что убедительно, что стоит добавить.

**Рефлексия**

Каждый участник — одно предложение: какая из 8 практик оказалась для вас самой ценной и почему?

</details>
