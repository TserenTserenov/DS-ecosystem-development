# Тарифы и пакеты

## 0. Назначение документа

Тарифная политика экосистемы. Два слоя: **подписка** (постоянная, для поддержания уровня) и **события** (on-demand, по необходимости). Тиры подписки совпадают с тирами платформы (T1–T4, [WP-74](../../0.OPS/0.9.Inbox/WP-74-platform-concept-of-use.md) §3). Вместо скидок — система баллов за активные действия.

**Семейство F4:** Целевая x Предприниматель — «Ценность и бизнес-модели»

---

## 1. Принципы ценообразования

### 1.1. Два слоя

| Слой | Что оплачивается | Модель | Аналоги рынка |
|------|------------------|--------|---------------|
| **Подписка** | Доступ к платформе, ИИ, ЦД, сообщество | Recurring, доступная | Coursera Plus ($59/мес), MasterClass ($10-20/мес) |
| **События** | Трансформация: практикумы, резидентуры, семинары | One-time, premium | Reforge ($2000/год), Maven ($500+/когорта) |

### 1.2. Ключевые принципы

1. **Подписка = инструменты**, события = трансформация. Наставник НЕ включён в подписку
2. **Баллы вместо скидок** — начисляются за активные действия, тратятся на события
3. **Прозрачность** — все цены публичны, нет скрытых платежей
4. **Доступность** — T2 от $15/мес, соответствует миссии «$10 и 10 часов в неделю»
5. **Pause-опция** — подписку можно поставить на паузу (1-3 мес) вместо отмены

## 3. События (Слой 2)

> Покупаются разово, поверх подписки. Можно оплатить баллами (полностью или частично).

### 3.1. Входные продукты (₽0 — ₽5000)

Цель: познакомить, вовлечь, показать маршрут. Человек ещё НЕ подписчик.

| Продукт | Цена | Формат | Что получает |
|---------|------|--------|-------------|
| **Вебинар** | ₽0 | Live 60-90 мин | Знакомство с подходом, живой контакт |
| **Марафон** | ₽0-1500 | 14 дней в боте, микро-задания | Ритм, первый опыт методики |
| **Диагностическая встреча** | ₽1500-5000 | 1:1, 30-60 мин | Разбор проблемы, персональный маршрут |

### 3.2. Программные продукты (on-demand)

Цель: навыковое обучение и создание шедевров. Для подписчиков T2+.

| Продукт | Формат (МИМ) | Длительность | Цена | Что получает |
|---------|-------------|--------------|------|-------------|
| **Практикум** | MIM.FMT.002 | 4-8 недель | **$300-500** | Навыковая прокачка через deliberate practice, ИИ-проверка ДЗ |
| **Резидентура** | MIM.FMT.003 | 6-12 недель | **$800-2000** | Шедевр с наставником, менторство, сертификат |
| **Семинар** | MIM.FMT.001 | 2-4 часа | **₽5000-12000** | Новое знание от эксперта, слайдомент, проблематизация |

### 3.3. Лестница продуктов

```
ВХОДНЫЕ (₽0-5000)          ПРОГРАММА ($300-2000)         УГЛУБЛЕНИЕ (₽5-12 тыс)
─────────────────────────── ────────────────────────────── ──────────────────────────
Вебинар (бесплатный)   ──► Практикум ($300-500)      ──► Семинар (₽5000-12000)

                            шедевр с наставником

           ▼                         ▼                          ▼
   «хочу попробовать»       «хочу научиться»           «хочу расширить фронтир»
   ещё не подписчик          подписчик T2+               подписчик T3-T4
```

## 4. Система баллов (вместо скидок)

### 4.1. Принцип

Вместо фиксированных скидок — начисление баллов за активные действия. Баллами можно полностью или частично оплачивать программные продукты (практикумы, резидентуры, семинары). Чем выше квалификация участника, тем больше множитель начисления.

### 4.2. Начисление баллов

| Действие | Баллы | Множитель по тиру |
|----------|-------|-------------------|
| Завершённый слот обучения | 5-10 | T2 x1, T3 x1.5, T4 x2 |
| Завершённый модуль | 20-50 | T2 x1, T3 x1.5, T4 x2 |
| Перенос в практику (артефакт) | 20-100 | T2 x1, T3 x1.5, T4 x2 |
| Peer-review чужой работы | 10-50 | T2 x1, T3 x1.5, T4 x2 |
| Вклад в сообщество (пост, кейс) | 10-30 | T2 x1, T3 x1.5, T4 x2 |
| Проверка ДЗ (наставник) | 5-20 | Фиксировано |
| Проект / рабочий продукт | 50-500 | T2 x1, T3 x1.5, T4 x2 |

### 4.3. Трата баллов (1 балл ≈ $1)

| Продукт | Эквивалент в баллах |
|---------|---------------------|
| Практикум ($300-500) | 300-500 |
| Резидентура ($800-2000) | 800-2000 |
| Семинар (₽5000-12000) | 50-120 |

> Баллы — признание вклада, а не покупаемая валюта. Мотивируют активное участие и апгрейд тира.

## 7. Путь участника: деньги и время

### 7.1. Три фазы становления созидателя

```
ФАЗА 1                    ФАЗА 2                        ФАЗА 3
Понять суть               Пройти программу               Поддерживать и развиваться
(1-3 мес)                 (9-18 мес)                     (бессрочно)
────────────────────────── ─────────────────────────────── ──────────────────────────►

Вебинар, марафон           T2/T3 подписка                  T2/T3 подписка
Диагностика                2-4 практикума                  1-2 семинара/год
T2 подписка                1 резидентура (опц.)            Сообщество + ЦД
                           Семинары по интересу
```

### 7.2. Расчёт для среднего участника

| | Фаза 1: Понять суть | Фаза 2: Программа | Фаза 3: Поддерживать |
|---|---------------------|-------------------|---------------------|
| **Длительность** | 1-3 мес | 9-18 мес | Бессрочно |
| **Время/неделю** | 5-10 ч | 10-15 ч | 3-5 ч |
| | | | |
| Входные | ~$30 | — | — |
| Подписка | $30 | $300-600 | $150/год |
| Практикумы | — | $800-1200 | $0-400 |
| Резидентура | — | $0-1200 | — |
| Семинары | — | $80-160 | $80-160 |
| | | | |
| **Итого деньги** | **~$60** | **$1180-$3160** | **$230-$710/год** |
| **Итого время** | **~100 ч** | **~650 ч** | **~200 ч/год** |

### 7.3. Сводка

| Метрика | Значение |
