# Продуктовые предложения: Программы развития

## 0. Назначение документа

Этот документ описывает **коммерческие предложения по программам развития** МИМ — каталог форматов, цены, условия, траектории участника.

**Семейство F4:** Целевая x Предприниматель — «Ценность и бизнес-модели»

**Связь с разделом 2.2:** Архитектура программ описана в [[О программах развития 2.2]], здесь — их ценность и коммерческие параметры.

> **Source-of-truth терминологии:** PACK-MIM. Терминология этого документа ДОЛЖНА соответствовать доменным сущностям PACK-MIM. Ключевые различения: MIM.D.001 (семинар != практикум), MIM.D.002 (практикум != резидентура), MIM.D.003 (резидентура != стажировка), MIM.D.005 (формат != программа).

---

## 1. Три программы и пять форматов

> **Различение MIM.D.005:** Программа = направленность развития (что развиваем). Формат = тип мероприятия (как устроено). Одна программа использует разные форматы.

### 1.1. Три программы развития

```
ЦЕПОЧКА РАЗВИТИЯ (MIM):

  Исследовательское (ИР)  →  Личное (ЛР)  →  Рабочее (РР)
  «Познакомиться»            «Набить руку»    «Выдать результат»
   MIM.PRG.003                MIM.PRG.001       MIM.PRG.002
```

| Программа | Что развиваем | Основной формат | Результат |
|-----------|---------------|-----------------|-----------|
| **Исследовательское развитие** (MIM.PRG.003) | Знание — доступ к новому | Семинар (MIM.FMT.001) | Коллективное понимание + слайдомент |
| **Личное развитие** (MIM.PRG.001) | Навык — мастерство инженера-менеджера | Практикум (MIM.FMT.002) | «Улучшенная версия себя» |
| **Рабочее развитие** (MIM.PRG.002) | Результат — рабочие успехи | Резидентура (MIM.FMT.003) | Шедевр (MIM.WP.002) |

### 1.2. Пять форматов мероприятий

| Формат | Главный эффект | Длительность (типовая) | Роли |
|--------|---------------|------------------------|------|
| **Семинар** (MIM.FMT.001) | Знакомство с новым знанием | 2-6 часов | Докладчик (MIM.R.002) + участники |
| **Практикум** (MIM.FMT.002) | Тренировка навыка (deliberate practice) | 6-12 недель | Наставник (MIM.R.001) + участники |
| **Резидентура** (MIM.FMT.003) | Рабочий результат (шедевр) | 6 недель | Наставник (MIM.R.001) + резидент (MIM.R.003) |
| **Разбор** (MIM.FMT.004) | Обратная связь + замер квалификации | 60-90 мин | Наставник + участники |
| **Марафон** (MIM.FMT.005) | Интенсивное погружение + ритм | 14 дней | Наставник (авто/ИИ) + участники |

> **Типичная ошибка MIM.FM.001:** «Послушал семинар — теперь умею.» Семинар знакомит, но не тренирует. Для навыка нужен практикум.

## 2. Семинары (программа исследовательского развития)

> **MIM.FMT.001 + MIM.PRG.003.** Цель — знакомство с новым, сложным, малодоступным материалом. Не тренировка навыка. Рабочий продукт — слайдомент (MIM.WP.001).

### 2.1. Каталог семинаров

| Семинар | Длительность | Формат | Цена |
|---------|--------------|--------|------|
| **Weekend: Системное мышление** | 2 дня (сб-вс) | Онлайн/офлайн | $149-299 |
| **Sprint Week: погружение в тему** | 5 дней | Онлайн | $299-499 |
| **Deep Dive: интенсив** | 3 дня | Офлайн | $499-799 |
| **Платные семинары (тематические)** | 2-6 часов | Онлайн | $49-149 |

### 2.2. Weekend-семинар: Системное мышление

**Цель:** Познакомиться с основами системного мышления. Понять подход и составить план развития.

**Расписание:**

| День | Время | Содержание |
|------|-------|------------|
| Суббота | 10:00-13:00 | Доклад: основы системного мышления (слайдомент) |
| | 14:00-17:00 | Пробная работа: моделирование своей задачи |
| | 17:00-18:00 | Q&A с докладчиком |
| Воскресенье | 10:00-13:00 | Пробная работа: решение проблемы |
| | 14:00-16:00 | Представление мини-результатов |
| | 16:00-17:00 | План развития: куда идти дальше (→ практикум) |

**Результат:** Первое знакомство + слайдомент для работы с ИИ-ассистентом + план перехода в практикум.

> **Важно:** Семинар НЕ даёт навык. После семинара участник направляется в практикум (MIM.D.001). Слайдомент можно сразу использовать с ИИ-ассистентом как мостик между «послушал» и «начал тренироваться».

## 3. Марафоны (вводный формат личного развития)

> **MIM.FMT.005.** Композитный формат: фиксированная по времени последовательность из уроков, заданий и практик с ежедневным ритмом. Точка входа в программу личного развития (MIM.PRG.001).

### 3.1. Каталог марафонов

| Марафон | Длительность | Формат | Цена |
|---------|--------------|--------|------|
| **Саморазвитие: первые шаги** | 14 дней | Онлайн (бот + Web App) | $99-199 |
| **Системное мышление: старт** | 14 дней | Онлайн (бот + Web App) | $99-199 |
| **Работа с ИИ: экзокортекс** | 14 дней | Онлайн (бот + Web App) | $99-199 |

### 3.2. Структура марафона

- Фиксированная длительность: 14 дней
- Ежедневный цикл: урок (семинарная часть) → задание (практикумная часть) → обратная связь (разбор)
- Трекинг прогресса через ИТ-платформу (бот / Web App)
- Наставничество: автоматизированное (ИИ-ассистент) + групповые разборы

> **Марафон != Курс.** Курс — дидактическая конструкция (downstream). Марафон — организационный формат, задающий ритм ежедневной практики.

## 4. Практикумы (программа личного развития)

> **MIM.FMT.002 + MIM.PRG.001.** Цель — «набить руку» в навыке через deliberate practice (MIM.M.001). Результат — «улучшенная версия себя», учебные проекты выбрасываются. Наставничество через разбор (MIM.M.002).

### 4.1. Каталог практикумов

| Практикум | Длительность | Ролевая траектория | Результат |
|-----------|--------------|-------------------|-----------|
| **Системное мышление** | 8 недель | Ученик | Навык моделирования, 3 учебных артефакта |
| **Работа с ИИ** | 6 недель | Ученик | Экзокортекс, 5 автоматизаций |
| **Управление знаниями** | 6 недель | Ученик | PKM-система, база знаний |
| **Проектное мышление** | 8 недель | Интеллектуал | 1 учебный проект |
| **Исследовательская работа** | 10 недель | Интеллектуал | Исследование + публикация |
| **Наставничество** | 12 недель | Профессионал | Готовность к роли наставника (MIM.R.001) |

> **Различение MIM.D.002:** Результат практикума — учебный (выбрасывается). «Смотри, как я изменился.» Для рабочего результата (шедевра) — резидентура.

### 4.2. Практикум «Системное мышление»

**Цель:** Освоить базовые инструменты системного мышления через deliberate practice (MIM.M.001). Автоматизировать базовые операции моделирования.

**Для кого:** Участники, прошедшие семинар или марафон по системному мышлению.

**Программа:**

| Неделя | Тема | Упражнение |
|--------|------|------------|
| 1 | Введение в системное мышление | Карта своей системы |
| 2 | Модели и моделирование | Модель проблемной ситуации |
| 3 | Причинно-следственные связи | Диаграмма причинных петель |
| 4 | Уровни абстракции | Многоуровневая модель |
| 5 | Эмерджентность и холизм | Анализ эмерджентных свойств |
| 6 | Системные архетипы | Определение архетипов в своей практике |
| 7 | Практика: решение проблемы | Системное решение реальной задачи |
| 8 | Разбор итогов | Демонстрация навыка |

**Формат:**
- 2 групповые сессии в неделю (90 мин)
- Еженедельное задание на границе возможностей (deliberate practice)
- Разбор (MIM.FMT.004) с наставником — обратная связь + замер квалификации (MIM.WP.003)
- Peer review в группе
- Безопасная среда, толерантная к ошибкам

**Цена:** $499 (со скидками подписчиков)

---

### 4.3. Практикум «Работа с ИИ»

**Цель:** Научиться использовать ИИ как инструмент мышления, а не замену ему. Настроить экзокортекс.

**Для кого:** Все, кто хочет грамотно интегрировать ИИ в свою работу.

**Программа:**

| Неделя | Тема | Упражнение |
|--------|------|------------|
| 1 | Модели ИИ и их возможности | Карта применений ИИ |
| 2 | Prompt engineering | Библиотека промптов |
| 3 | Экзокортекс: структура | Настроенный экзокортекс |
| 4 | Автоматизации и workflows | 3 автоматизации |
| 5 | ИИ для исследований | Исследование с ИИ |
| 6 | Этика и ограничения | Персональная политика ИИ |

**Цена:** $399

---

### 4.4. Практикум «Наставничество»

**Цель:** Подготовиться к роли наставника (MIM.R.001) — научиться вести разборы (MIM.FMT.004) и давать обратную связь через review (MIM.M.002).

**Для кого:** Профессионалы и Исследователи, готовые передавать мастерство.

**Программа:**

| Неделя | Тема | Практика |
|--------|------|----------|
| 1-2 | Основы наставничества | Наблюдение за наставником |
| 3-4 | Обратная связь через разбор (MIM.M.002) | Практика peer feedback |
| 5-6 | Ведение групп | Со-ведение разбора |
| 7-8 | Индивидуальная работа | Ведение 1 участника |
| 9-10 | Сложные ситуации | Разбор кейсов |
| 11-12 | Квалификация | Самостоятельное ведение разбора |

**Результат:** Функциональная квалификация наставника (MIM.R.001) в экосистеме, подтверждённая замерами (MIM.WP.003).

**Цена:** $799

## 5. Резидентуры (программа рабочего развития)

> **MIM.FMT.003 + MIM.PRG.002.** Цель — создание шедевра (MIM.WP.002): внешне предъявляемого рабочего результата с заранее заданными критериями. Проект свой (не выданный начальником). Наставничество через review (MIM.M.002).

> **Различение MIM.D.003:** Резидентура != стажировка. В резидентуре проект свой, инициатива у резидента. В стажировке проект выдан начальником.

### 5.1. Структура в МИМ

- 10 резидентур по 6 недель каждая
- Всего ~600 часов
- Строго последовательный порядок при первом прохождении
- Можно перепроходить в произвольном порядке потом (MIM.FM.003: не «оставаться на второй год»)

### 5.2. Коммерческие модели резидентур

| Модель | Суть | Экономика | Для кого |
|--------|------|-----------|----------|
| **С наставничеством** | Резидент создаёт шедевр под наставничеством | Участник платит $299-799 за резидентуру | Профессионалы |
| **Проектная** | Шедевр = проект для экосистемы | Revenue share 20-40% | Профессионалы с опытом |
| **Исследовательская** | Шедевр = вклад в фронтир знаний | Грант $500-2000/мес | Исследователи |

### 5.3. Резидентура с наставничеством (основная)

**Как это работает:**

1. **Определение шедевра** — резидент сам выбирает проект (MIM.D.003: свой проект)
2. **Договорённость о критериях** — наставник и резидент согласуют критерии шедевра заранее
3. **Работа** — резидент работает 6 недель, приносит промежуточные результаты на разбор
4. **Еженедельные разборы** (MIM.FMT.004) — наставничество через review (MIM.M.002) + замер квалификации (MIM.WP.003)
5. **Результат** — шедевр (MIM.WP.002), предъявляемый внешнему миру

### 5.4. Проектная резидентура

**Шедевр = проект для экосистемы.** Резидент выбирает проект из списка или предлагает свой.

**Примеры шедевров:**
- Разработка нового руководства (30% от продаж)
- Создание практикума для новой области (25% от выручки)
- Развитие ИИ-системы платформы (20% от подписок)
- Проект для внешнего заказчика (40% от оплаты)

### 5.5. Исследовательская резидентура

**Шедевр = вклад в развитие фронтира знаний.**

**Формат:**
- Грант: $500-2000/мес в зависимости от scope
- Наставническая поддержка через разборы (MIM.M.002)
- Доступ к ресурсам экосистемы
- Публикация результатов

**Темы исследований (примеры):**
- Метрики развития системного мышления
- Эффективность ИИ-ассистентов в обучении
- Модели peer learning
- Экономика баллов в образовательных экосистемах

## 6. Путь по программам

### 6.1. Рекомендованная траектория

> Цепочка: Исследовательское → Личное → Рабочее (MIM). Семинар → Практикум → Резидентура.

```
РОЛЕВАЯ             ФОРМАТ                               РЕЗУЛЬТАТ
ТРАЕКТОРИЯ
───────────────────────────────────────────────────────────────────────

Ученик        ───>  Семинар / Марафон «Введение»     ──> Знакомство
                    |
              ───>  Практикум «Системное мышление»    ──> Навык моделирования
                    |
              ───>  Практикум «Работа с ИИ»           ──> Экзокортекс

Интеллектуал  ───>  Практикум «Управление знаниями»   ──> PKM-система
                    |
              ───>  Практикум «Проектное мышление»    ──> Навык проектной работы

Профессионал  ───>  Резидентура (шедевр)              ──> Рабочий результат
                    |
              ───>  Практикум «Наставничество»        ──> Роль наставника (R.001)

Исследователь ───>  Резидентура исследовательская      ──> Вклад в область

Мастер (R.004) ──>  Резидентуры (перепрохождение)     ──> Мастерство в реальных проектах
```

### 6.2. Комбинации и пакеты

| Комбинация | Стоимость | Экономия | Для кого |
|------------|-----------|----------|----------|
| Практикум «Системное мышление» + «Работа с ИИ» | $799 | $99 (11%) | Ученики |
| Практикум «Проектное мышление» + «Управление знаниями» | $799 | $99 (11%) | Интеллектуалы |
| 3 любых практикума | $1199 | $198 (14%) | Активные участники |
| Все 6 практикумов | $2199 | $694 (24%) | Максимальное развитие |
| Марафон + 1 практикум | от $499 | ~15% | Новички |
