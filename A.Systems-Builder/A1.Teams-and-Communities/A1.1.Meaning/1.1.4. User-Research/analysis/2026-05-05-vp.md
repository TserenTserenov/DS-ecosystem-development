# Анализ интервью ВП (2026-05-05) — C2 Долгосрочный подписчик / IWE-non-adopter

## D1 Портрет

- МФТИ — физтех; учёный → предприниматель → AI builder; строит продукты с AI под капотом руками сам (разработка + маркетинг)
- Активные проекты: (1) AI-аналитик для крупного химического производства — «боксы» молекул для привлечения инвестиций; (2) ассистент-второй пилот для языковых школ и репетиторов
- Участвует в хакатонах; собрал собственную группу для совместного изучения системного мышления
- Постоянный подписчик (~2 года, перманентная цена); прошёл практики ЛР + часть рациональной работы

## D2 JTBD

- «Выстроить актив на пересечении физического и цифрового мира, пока ИИ не разобрались все» — стать имплементатором AI-технологий лабораторий в реальный бизнес
- «Оперировать на более высоких системных уровнях» — системное мышление как инструмент масштабирования: выходить на более крупные вопросы и проекты
- «Сделать собственную систему хранения и обзора знаний» — уже реализовано через Obsidian + Code + Whisper + еженедельный AI-обзор

## D3 Боль / Неудовлетворённость

- «Я вижу, что через год-два скиллы сгорят» — острое ощущение устаревания компетенций → давление на срок построения актива
- «Я сильно недоиспользую возможности ИИ» — ощущает потолок, не видит как его пробить без системного уровня
- «Не уловил для себя ценности» IWE — подписчик с 2 летним стажем, изучает системное мышление, но IWE прошёл мимо
- «Некоторая категоричность» в постах/комментариях ШСМ — отталкивает, держит дистанцию

## D4 Методы (что пробовал)

- **Code + Codex** как основные среды (не VS Code / Cursor; отличие — собственные модели/среды без extension-IDE подхода)
- Собственная PKM-архитектура: Obsidian Vault + векторная БД + Whisper → сырые заметки → атомарные заметки
- Еженедельный AI-обзор репозиториев и заметок: «часто не замечаю сколько сделал, помощник показывает»
- Ежемесячная автоматическая пересборка линков / перепросмотр связей
- Совместное изучение системного мышления в группе (организовал сам)
- Хакатоны, в том числе у сторонних организаторов

## D5 Альтернативы

- Stanford AI Vision курс (топ-спикеры OpenAI, Nvidia и др.) — для понимания рынка
- Хакатонная культура — практика и нетворк
- Собственная PKM-система — уже работающая альтернатива IWE-подходу (без PAK, без ОРЗ формально, но схожая функция)

## D6 Сегмент

C2 — долгосрочный подписчик MIM (2 года, перманентная цена). IWE-non-adopter: несмотря на активную AI-практику и подписку, IWE не попробовал. Критически важный сегмент: человек, который уже платит, уже в теме, но не использует ключевой продукт. Разрыв между «быть подписчиком MIM» и «использовать IWE».

## D7 Онбординг (барьеры)

- **Нет понятного входного сценария для практикующего AI builder'а**: уже есть своя PKM (Obsidian + Code), уже есть workflow — не понятно, что IWE добавляет поверх
- Отсутствие сценария «у тебя уже есть система — вот как IWE её расширяет или заменяет»
- Рациональную работу прошёл частично, решил достаточно — нет механизма возврата / дозирования для «уже знающего»
- Категоричность сообщества — психологический барьер для активного участия

## D8 Доверие (откуда узнал, кому доверяет)

- Первый контакт — курсы в МФТИ (системное мышление, системная инженерия, системная архитектура) → зацепило идеями
- Доверяет «встроенной логике»: «то, что пропагандируют мимы — вся встроенная логика — мне прям сильно импонирует»
- Собрал группу сам → уже рекомендатор, но пассивный

## D9 Ожидания

- Через 1–2 года: выстроенный актив на стыке физического и цифрового; первые устойчивые клиенты
- Текущий фокус: закрыть проект с химическим производством, привлечь инвестиции
- От ШСМ: паттерны мышления для более высоких системных уровней → масштаб задач

## D10 Ценностное предложение (что главное для него)

- «Заскочить в вагон, пока все с ним не разобрались» — конкурентное окно; AI как имплементатор, не потребитель
- Встроенная когерентность: не набор приёмов, а система с внутренней логикой
- Долгосрочный актив, не быстрый доход

## D11 Монетизация

- Постоянная подписка, перманентная цена — «не обременительна»; привязана не к ценности в моменте, а к долгосрочной принадлежности
- Психология: «могу месяц не открывать — окей, потом открою»; не активный потребитель, но и не уходит
- Ценовой порог: в 3–5 раз вырастет → перейдёт на ситуативные покупки (на период совместной учёбы группы)

## D12 Технический барьер

- **Низкий с точки зрения доступа** (в отличие от ИО/ИК): нет проблем с VPN, нет ограничений
- Использует Code + Codex (не Claude Code / Cursor) — другая экосистема; IWE заточен под VS Code + Claude Code → смена инструмента как барьер
- Убеждение: «Code из коробки закрывает архитектурные вопросы прекрасно» — IWE воспринимается как дублирующий уровень, а не добавляющий

## D13 Язык (дословные цитаты)

- «Я вижу глобальное недоиспользование ИИ, и хочу заскочить в этот вагон, пока все с ним не разобрались»
- «Через год-два скиллы сгорят»
- «Встроенная логика» — ключевой аттрактор
- «Некоторая категоричность» — ключевой репеллент
- «Пока что я не уловил для себя ценности» — про IWE
- «Часто не замечаю и не понимаю, что я действительно много сделал — а ощущение, что ничего. Сделал помощника»
- «Быть имплементатором технологий крупных лабораторий в реальный бизнес»

## D14 R2-сигналы

- **Высокий потенциальный R2**: уже организовал группу для совместного изучения системного мышления — это сильнейший R2-сигнал, ещё не конвертированный
- **Текущий R2 пассивный**: общается с Иваном [Р.], Романом, Ильёй [Л.] — точечная сеть, не активное продвижение
- Красный флаг категоричности тормозит публичную активность («держал себя за хвост, чтобы не высказаться»)

## D15 Мировоззрение (ступень по PD.FORM.080)

**Ступень 3–4 — «Окружение влияет на меня» → «Мир — система»**. Видит макро-системы: химическое производство, рынок ИИ, цивилизационный сдвиг. Осознаёт необходимость более высоких системных уровней мышления. Строит группу сопроизводителей сам — это признак ст. 3+. Знает, что скиллы устаревают и формирует долгосрочный актив — признак стратегического горизонта. Мировоззренческий сдвиг уже произошёл; методы M1/M2 не формализованы (IWE не используется), но функционально реализованы своими инструментами.

## D16 Слоты мастерства

- **W**: высокое — AI builder, строит продукты руками, нет проблем с рабочим ритмом
- **M1**: высокое неформально — еженедельный AI-обзор, ежемесячная пересборка знаний; система сделана своими руками
- **M2 (IWE/ОРЗ)**: нулевое формально — не использует IWE; но функциональный аналог PKM уже работает
- **M4**: формируется — идёт на системное мышление; строит агентов и «боксы молекул» как системные объекты
- **IT**: высокое — разработчик, понимает архитектуру, строит AI-системы с нуля
- **A**: низкое публично — есть группа, но нет публичных кейсов/выступлений

## D17 Характеристики выпускника

- **Ясность**: высокая относительно профессионального направления; строит конкретный продукт с конкретным покупателем
- **Собранность**: высокая — система PKM работает, делает много руками; замечает продуктивность через AI-помощника
- **Способность производить**: высокая — AI аналитик, ассистент для языковых школ — реальные продукты
- **Стрессоустойчивость**: высокая — переход от управленца к исполнителю-builder; работает в условиях неопределённости

## D18 Форма vs Содержание

Форма: реализована самостоятельно (собственный ритм, PKM, AI-обзор) без ШСМ-методологии формально. Содержание: экспертиза AI-интеграции в физический бизнес высокая. ШСМ нужен ВП для расширения системного горизонта, а не для построения формы — форма уже есть. Это нетипичный профиль: большинство пользователей приходят без формы, ВП приходит с формой, хочет расширить содержательный горизонт.
