# Реестр задач из интервью

> **Source-of-truth** по задачам, извлечённым из интервью с пользователями. Инвариант: ни одна задача не теряется — у каждой строки либо ссылка на РП, либо явное `rejected: <причина>`.

> **Жизненный цикл строки:** `pending-review` → (R14 решение) → `routed: WP-NNN-Ф{X}` либо `routed: new WP-NNN` либо `rejected: <причина>`. После закрытия РП — пометка `✅ implemented` с датой.

## Колонки

| Поле | Описание |
|------|----------|
| # | порядковый номер записи |
| Дата | когда задача добавлена в реестр |
| Интервью | имя файла транскрипта (без расширения) |
| Направление | код из § 5 process.md (D1-D14) |
| Задача | формулировка действия |
| Статус | `pending-review` / `routed` / `rejected` / `implemented` |

| Заметки | дополнительный контекст |
