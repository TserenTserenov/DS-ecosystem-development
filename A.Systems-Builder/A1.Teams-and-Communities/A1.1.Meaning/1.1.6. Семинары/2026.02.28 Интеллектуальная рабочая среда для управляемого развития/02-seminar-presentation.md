## Слайд 11: 4 архитектурных вида IWE

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│     IWE — ИНТЕЛЛЕКТУАЛЬНАЯ РАБОЧАЯ СРЕДА                 │
│     4 архитектурных вида (ISO 42010)                    │
│                                                         │
│   ┌──────────────────┐  ┌──────────────────┐            │
│   │   СИСТЕМЫ        │  │   ЗНАНИЯ         │            │
│   │   (U.System)     │  │   (U.Episteme)   │            │
│   │                  │  │                  │            │
│   │ Claude Code      │  │ Экзокортекс      │            │
│   │ Бот @aist_me_bot │  │ Pack (знания)    │            │
│   │ MCP-серверы      │  │ Методология      │            │
│   │ Git, WakaTime    │  │ (протоколы, FPF) │            │
│   └──────────────────┘  └──────────────────┘            │
│                                                         │
│   ┌──────────────────┐  ┌──────────────────┐            │
│   │   РОЛИ           │  │   АРТЕФАКТЫ      │            │
│   │ (U.RoleAssign.)  │  │   (U.Work)       │            │
│   │                  │  │                  │            │
│   │ Стратег          │  │ DS-strategy      │            │
│   │ Экстрактор       │  │ Pack-репозитории  │            │
│   │ Синхронизатор    │  │ Цифровой двойник │            │
│   │ Агент разверт.   │  │ Планы, отчёты    │            │
│   └──────────────────┘  └──────────────────┘            │
│                                                         │
│     IWE ≠ экзокортекс.                                  │
│     Экзокортекс — это знания (один из четырёх видов).    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Время:** 4 мин
**Скрипт:** «IWE — это не один инструмент. Это среда, описываемая через четыре архитектурных вида. Системы — это конкретные программы и сервисы. Знания — экзокортекс, Pack, методология. Роли — кто что делает (Стратег планирует, Экстрактор извлекает знания). Артефакты — рабочие продукты, которые создаются.
