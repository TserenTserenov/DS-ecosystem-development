# Чек-лист развёртывания IWE

> Раздаточный материал для участников семинара «От экзокортекса к интеллектуальной рабочей среде» (28 февраля 2026)

---

## Что вы получите

Работающую интеллектуальную рабочую среду (IWE) уровня 1:
- Экзокортекс с 3 слоями памяти (MEMORY.md + CLAUDE.md + memory/)
- ИИ-ассистент (Claude Code), который помнит ваши правила и задачи
- Бот @aist_me_bot в Telegram — экзокортекс с телефона
- Цикл сессии ОРЗ (Открытие → Работа → Закрытие)

**Время:** один вечер (2–3 часа)

---

## Перед началом

**Что нужно:**
- Компьютер (macOS, Linux или Windows с WSL)
- Стабильный интернет
- Подписка Claude Pro или Max ($20+/мес) — [claude.ai](https://claude.ai)

**Не нужно:**
- Уметь программировать
- Разбираться в Git (научитесь по ходу)

---

## 7 шагов

### Шаг 1. Установить VS Code (~10 мин)

1. Скачайте VS Code: [code.visualstudio.com](https://code.visualstudio.com)
2. Установите и запустите
3. Рекомендуемые расширения (не обязательно сразу):
   - **Markdown All in One** — удобная работа с .md файлами
   - **GitLens** — визуализация истории

> VS Code — ваше рабочее пространство. Здесь вы будете редактировать память, читать знания, работать с Claude.

### Шаг 2. Настроить Git + GitHub (~15 мин)

1. **Создайте аккаунт GitHub:** [github.com](https://github.com) (если нет)

2. **Установите Git:**
   - macOS: `xcode-select --install` (в Терминале)
   - Linux: `sudo apt install git`
   - Windows: [git-scm.com](https://git-scm.com)

3. **Настройте имя и email:**
   ```bash
   git config --global user.name "Ваше Имя"
   git config --global user.email "ваш@email.com"
   ```

4. **Установите GitHub CLI:**
   - macOS: `brew install gh`
   - Linux: см. [cli.github.com](https://cli.github.com)
   - Авторизуйтесь: `gh auth login`

> Git — система версий. Все ваши знания и память хранятся в Git-репозиториях. Это надёжнее любого облака.

### Шаг 3. Установить Claude Code (~5 мин)

1. **Установите Node.js** (если нет): [nodejs.org](https://nodejs.org) (версия LTS)

2. **Установите Claude Code:**
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

3. **Проверьте установку:**
   ```bash
   claude --version
   ```

4. **Авторизуйтесь:** при первом запуске `claude` он попросит войти в аккаунт

> Claude Code — ваш ИИ-ассистент в терминале. Он читает CLAUDE.md и MEMORY.md автоматически. Это не ChatGPT — это среда.

### Шаг 4. Развернуть экзокортекс из шаблона (~10 мин)

1. **Склонируйте установщик и запустите setup:**
   ```bash
   cd ~/Github    # или ваша рабочая директория из Шага 0
   gh repo clone TserenTserenov/DS-ai-systems -- --depth 1
   cd DS-ai-systems/setup
   bash setup.sh
   ```

2. **Скрипт спросит:**
   - GitHub username
   - Рабочую директорию (по умолчанию `~/Github`)
   - Путь к Claude CLI (определяется автоматически)
   - Часовой пояс для Стратега (UTC)

3. **Что делает setup.sh:**
   - Форкнет шаблон FMT-exocortex-template в ваш GitHub
   - Заменит плейсхолдеры на ваши данные
   - Скопирует CLAUDE.md и memory/ в нужные места
   - Установит Стратега (автоматическое планирование)
   - Создаст DS-strategy/ — приватный репозиторий для стратегирования

4. **Проверьте результат:** у вас должна появиться папка с экзокортексом и DS-strategy/

> Шаблон — стартовая точка. Стандартные протоколы и роли уже настроены. Ваши личные данные (память, знания) — только ваши.

### Шаг 5. Заполнить MEMORY.md (~15 мин)

Откройте файл `MEMORY.md` в вашем экзокортексе (VS Code) и заполните:

```markdown
# Оперативная память

## Текущие задачи
- [ ] Задача 1 (опишите, над чем работаете)
- [ ] Задача 2

## Цели
- Ваша главная цель обучения/развития
- Ваш текущий метод (как вы работаете)

## Уроки
- Что вы уже знаете о своём подходе к работе
```

**Подсказки:**
- Пишите как для себя через неделю: «что на столе прямо сейчас?»
- Не пишите много — ≤100 строк. Это оперативная память, не дневник
- Можно вернуться и дополнить позже

> MEMORY.md загружается ВСЕГДА, когда вы работаете с Claude Code. Это то, что ИИ «видит» о вас в каждой сессии.

### Шаг 6. Первая сессия по ОРЗ (~30 мин)

Откройте Claude Code в папке вашего экзокортекса и попробуйте полный цикл:

**Открытие:**
```
Вы: "Помоги мне [ваша задача]"
Claude: Проверит план (WP Gate) → объявит роль → спросит подтверждение
```

**Работа:**
```
Работайте над задачей вместе с Claude.
На каждом рубеже Claude спросит: «Есть знание для записи?»
```

**Закрытие:**
```
Вы: "Закрывай"
Claude: Зафиксирует результат → обновит память → запушит
```

> Это цикл ОРЗ (Открытие → Работа → Закрытие). Каждая сессия начинается с проверки плана и заканчивается фиксацией результата. Контур замкнут.

### Шаг 7. Подключить бота @aist_me_bot (~5 мин)

1. Найдите в Telegram: **@aist_me_bot**
2. Нажмите `/start`
3. Попробуйте:
   - `/plan` — ваш план на сегодня
   - `.заметка` — быстрая запись мысли (точка + текст)
   - Задайте вопрос — бот ответит, опираясь на базу знаний
   - `/test` — тест по вашей теме обучения

> Бот — вход в экзокортекс с телефона. Пришла мысль в метро → записал за 3 секунды. Не потеряется.

## Помощь

- **Бот не отвечает?** Напишите `/start` заново
- **Claude Code не видит MEMORY.md?** Проверьте, что запускаете `claude` из папки экзокортекса
- **setup не сработал?** Повторите шаг 4 или напишите в сообщество
- **Вопросы по содержанию?** Загляните в LEARNING-PATH.md
