# Разделение контуров Россия/мир (Regional Split)

> **Source-of-truth (решения):** [WP-215](../../../../0.OPS/0.9.Inbox/WP-215-infra-russia-world-split.md)
>
> Здесь -- архитектурная проработка разделения по трём уровням. Доменные решения (SC.105, экономика вклада) -- в Pack. Реализационные -- здесь.

## Назначение

Проработка трёхуровневого разделения экосистемы Aisystant на два контура (Россия и мир) с сохранением единого сообщества. Три уровня -- это три ортогональных среза одной задачи.
