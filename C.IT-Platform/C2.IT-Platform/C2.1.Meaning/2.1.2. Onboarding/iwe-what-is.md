# Что такое IWE — и зачем он вам?

> Черновик для холодного канала (лендинг WP-188). Source-of-truth → после валидации Г-К1 мигрирует в PACK-digital-platform.

## Проблема, которую решает IWE

ИИ-ассистенты стали мощнее — но большинство пользователей сталкиваются с одним и тем же:

- **Контекст теряется.** Каждая новая сессия с ИИ — с чистого листа. Вчерашние решения и договорённости забыты.
- **Знания остаются в голове.** Прошли курс, решили задачу — но через месяц не можете воспроизвести ход мысли.
- **ИИ заменяет мышление, а не усиливает его.** Вы получаете ответ, но не становитесь компетентнее. Без ИИ — снова на нуле.
- **Нет системы.** Планы — в заметках, задачи — в голове, знания — в чатах. Всё разрозненно.

## Что такое IWE

**IWE (Intellectual Work Environment)** — интеллектуальная рабочая среда. Аналог IDE, но для мышления, а не для кода.

Как IDE объединяет редактор, компилятор и дебаггер — IWE объединяет знания, планирование и ИИ-агентов в одну среду:

| IDE (для кода) | IWE (для мышления) |
|---------------|-------------------|
| Редактор → пишешь код | Экзокортекс → фиксируешь знания и контекст |
| Компилятор → проверяет синтаксис | Принципы и протоколы → проверяют корректность решений |
| Дебаггер → находит ошибки | АрхГейт → оценивает архитектурные решения |
| Git → история изменений | Стратег → история работы и планирование |

## Чем IWE отличается от ChatGPT и Cursor

| | ChatGPT / Claude.ai | Cursor | IWE |
|---|---|---|---|
| **Контекст** | Теряется между сессиями | Только код проекта | Сохраняется бессрочно в вашем репо |
| **Знания** | В модели (не ваши) | В файлах проекта | В персональных Pack-базах (ваши) |
| **Развитие** | Нет | Нет | ОРЗ-протоколы формируют мышление, а не просто дают ответы |
| **Тип работы** | Любые запросы | Разработка | Интеллектуальная работа: стратегия, планирование, развитие |

## Ключевой принцип: экзоскелет, не протез

IWE усиливает ваше мышление, а не заменяет его. После каждой сессии вы становитесь компетентнее — а не просто получаете результат, который забудется через неделю.

## Для кого IWE

IWE подходит, если вы:
- Ведёте интеллектуальную работу (стратегия, исследования, разработка продукта, обучение)
- Хотите, чтобы ИИ работал с вашим контекстом, а не с чистого листа
- Готовы вложить 1–2 недели на настройку среды, чтобы получить годы эффекта

IWE **не подходит**, если вы ищете чат-бот для разовых запросов — для этого есть ChatGPT.
