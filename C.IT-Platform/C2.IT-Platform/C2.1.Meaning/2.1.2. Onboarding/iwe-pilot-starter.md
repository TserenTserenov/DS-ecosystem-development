# Первые шаги пилота IWE

> Этот документ — стартовая инструкция для пилотов программы развития до готовности персональных руководств (Портной WP-222).
> После готовности Портного этот гайд заменяется персональным руководством через `/personal-guide-start`.

## Добро пожаловать в программу пилота

Вы входите в первую когорту пользователей IWE. Ваша задача — за первые 7 дней:
1. Установить среду
2. Пройти ОРЗ-ритуал три раза
3. Дать обратную связь в боте

## Что вы получаете как пилот

| Что | Где |
|-----|-----|
| Персональная среда IWE (VS Code + Claude Code) | На вашем компьютере |
| Aisystant бот | Telegram [@aist_me_bot](https://t.me/aist_me_bot) |
| Стратег-агент | В VS Code — агент для планирования и ревью |
| Поддержка пилота | Через бот (`/feedback`) или в чате когорты |

## Шаг 0: Убедитесь, что у вас есть

- [ ] Компьютер (macOS, Linux, Windows/WSL)
- [ ] VS Code установлен
- [ ] Аккаунт Aisystant активирован (письмо от бота)
- [ ] GitHub-аккаунт (нужен для персональных репозиториев)

## Шаг 1: Установка Claude Code (~10 мин)

```bash
npm install -g @anthropic-ai/claude-code
claude --version  # должно вернуть версию
```

Если npm не установлен — сначала установите Node.js (nodejs.org).

## Шаг 2: Форк шаблона IWE (~5 мин)

```bash
# Форкните FMT-exocortex-template на GitHub (кнопка Fork)
# Затем:
git clone https://github.com/<ваш-логин>/FMT-exocortex-template ~/IWE
cd ~/IWE
```

## Шаг 3: Запустите первый Day Open (~5 мин)

```bash
cd ~/IWE
claude
```

В агенте напишите:
```
открывай
```

Агент соберёт ваш контекст и составит план дня. Если сегодня первый день — он создаст базовую структуру.

## Шаг 4: Создайте свой первый Pack (~15 мин)

Pack — ваша база знаний. Начните с простого: напишите агенту:

```
создай Pack для [ваша предметная область]
```

Например: "создай Pack для менеджмента продукта" или "создай Pack для системного мышления".

## Шаг 5: Закройте день

```
закрывай
```

Агент зафиксирует что сделали, обновит планы.

## Расписание первой недели

| День | Фокус | Время |
|------|-------|-------|
| 1 | Установка + первый Day Open | 30 мин |
| 2 | Работа в ОРЗ-ритуале (открыл → поработал → закрыл) | 2× по 45 мин |
| 3–4 | Создание первого Pack | 1–2 часа |
| 5 | Day Open + Day Close привычным ритмом | 2× по 45 мин |
| 6–7 | Первый Week Close — агент соберёт итоги | 30 мин |

## Обратная связь (обязательна для пилотов)

После первых 7 дней напишите в бот:

```
/feedback
```

Ваши наблюдения критически важны — они формируют следующую итерацию.

**Что особенно ценно:**
- Что было непонятно при установке?
- Где ОРЗ-ритуал казался лишним?
- Какую задачу вы хотели решить, но не смогли?

## Что будет дальше

После того как мы обработаем обратную связь первых пилотов, вы получите **персональное руководство** — сгенерированное на основе вашего уровня, домена и целей. Это уже не статичный гайд, а среда, настроенная под вас.
