## 1. Onliness Statement

> Формат Neumeier (*The Brand Gap*): единственный [категория], который [дифференцирующая ценность], для [аудитория] с [потребность / JTBD].

**Draft v0.1:**

> IWE — единственная рабочая среда для созидателя, где открытая методология системного мышления встроена в инструмент, а не передаётся через учебник перед работой: человек осваивает методологию в деле, а среда растёт вместе с ним — не усредняет под большинство.

**Компоненты:**

| Компонент | Содержание |
|-----------|-----------|
| Категория | Рабочая среда (Intellectual Work Environment) |
| Для кого | Созидатель — человек, который строит системы профессионально и работает с ИИ |
| Чем отличается | Методология встроена в инструмент (а не передаётся до работы) |
| Уникальная ценность | Человек учится через дело; среда растёт вместе с ним |
| Противопоставление | Не ещё один ИИ-ассистент, который отвечает на упрощённую версию вопроса |

**Для EN-аудитории (YC):**

> IWE is the only intellectual work environment where an open systems-thinking methodology is embedded in the tool, not taught before using it — so creators master it through real work, and the environment evolves with them.

**Примечание:** onliness требует валидации на 5+ холодных интервью. До результатов — рабочая гипотеза.

## 2. Четыре природы IWE

> Смысловая рамка для дизайнера, копирайтера, маркетолога. Из этих природ выведены формулировки, архетип и визуальная метафора. Источник: `Уникальность IWE.md §«Четыре природы»`.

| Природа | Что даёт пользователю | Следствие для бренда |
|---------|----------------------|----------------------|
| **Наставник** | Среда знает, где ты в развитии, адаптируется под тебя | «Растёт вместе с вами» — не метафора, а техническая реальность |
| **Мастерская** | Рабочий стол для дела, не учебный класс | Акцент на «воплощение», не «изучение» |
| **Сотворец** | Пользователь вкладывает знания, среда возвращает структурированный опыт | Ко-эволюция: оба меняются |
| **Аватар** | Личный IWE включён в сеть сообщества и методологии | Узел коллективного интеллекта, не изолированный контур |

## 3. Бренд-архетип

> По системе Кэрол Пирсон (12 архетипов). Архетип задаёт тональность, визуальный язык, нарратив.

### Рекомендация: Sage (Мудрец) + Creator (Творец)

**Sage — первичный:**

| | |
|---|---|
| Движущая сила | Поиск истины и понимания |
| Главный страх | Быть введённым в заблуждение, остаться в невежестве |
| Стратегия | Накапливать знание; самопознание через рефлексию |
| Темп | Вдумчивый, точный, методичный |
| Примеры брендов | Google, Harvard, McKinsey, National Geographic, MIT |

Sage подходит IWE потому что:
- Ядро — FPF-методология (11 лет живой практики системного мышления)
- Продукт передаёт мудрость через опыт, не через инструкцию
- Аудитория ценит интеллектуальную точность выше «вау-эффекта»

**Creator — вторичный:**

| | |
|---|---|
| Движущая сила | Создать нечто со значением, выдержавшее время |
| Главный страх | Быть посредственным или лишённым видения |
| Стратегия | Развивать мастерство и художественный контроль |
| Темп | Созидательный, ремесленный, вдохновляющий |
| Примеры брендов | Apple, Lego, Adobe |

Creator подходит потому что:
- IWE — мастерская (рабочий стол), не читальный зал
- Пользователь приходит создавать системы, а не потреблять контент
- Слоган «вместе с ИИ вы воплощаете» — Creator-нарратив

**Комбинация Sage + Creator даёт:** «Мудрый мастер» — как лучший столярный инструмент, созданный с глубоким знанием ремесла.

### Альтернативы для обсуждения

| Архетип | Аргумент ЗА | Аргумент ПРОТИВ |
|---------|------------|----------------|
| **Magician (Маг)** | Обещает трансформацию: «проблема → задача». Акцент на превращении хаоса в ясность | Склоняет к громким обещаниям. Риск восприниматься как «лайфхак» или «магическое мышление» |
| **Explorer (Искатель)** | «Рост через исследование» — подходит к персональному развитию | Слишком ненаправленно. IWE — не о свободном блуждании, а о движении по траектории |

**Вывод:** Sage + Creator — наилучшее сочетание для 11-летней методологии с рабочей средой. Magician — рассмотреть как усиление тональности для конкретных кампаний (когда нужен трансформационный нарратив), но не как основу идентичности.

## 4. Целевая аудитория

> Первичный портрет. Служба продвижения адаптирует под сегменты C1-C4 (→ Приложение 1. Сегменты и валидация).

### Первичная аудитория: Созидатель в эпоху ИИ

**Кто:**
- Профессионал, работающий с знанием: инженер, продакт, предприниматель, стратег, методолог, руководитель
- Уже использует ИИ-инструменты в работе
- Строит системы (продукты, организации, процессы, стратегии) — профессионально или как основной смысл
- 28-45 лет, образование выше среднего, склонность к рефлексии

**Боли:**
- «ИИ отвечает на упрощённую версию моего вопроса, а не на исходную проблему»
- «Чувствую, что теряю конкурентное преимущество — ИИ делает за час то, что раньше я делал за неделю»
- «Работаю 12 часов, вывод всё равно пропорционален моему личному времени»
- «Нет среды, которая помнит мои проекты и развивается вместе со мной»

**JTBD (Jobs-to-be-Done):**
> Нанимаю IWE не чтобы «помогли писать», а чтобы «дали партнёра, который глубоко понимает моё дело, помогает мне расти и помнит мой контекст годами».

**Что НЕ ищет:**
- Ещё один ИИ-чат или ассистент
- Курс, который нужно пройти до работы
- Инструмент, который замещает его мышление

### Вторичная аудитория: Международный основатель (Track B / YC)

- Строит продукт или компанию в области knowledge work / AI
- Ищет экзоскелет для работы команды — не просто LLM
- Видит дефицит «methodological depth» в существующих AI tools
- Язык: EN (→ Ф2 имя + Ф3 manifesto EN)

## 6. Что НЕ является IWE (антипозиционирование)

> Важно для дизайнера и копирайтера: что отрицает бренд.

- ❌ «Ещё один ИИ-ассистент» — IWE не диалог, а рабочая среда с памятью и методологией
- ❌ «Курс» или «образовательная платформа» — не учим до работы, даём среду для работы
- ❌ «Автопилот для продуктивности» — не берём мышление на себя; усиливаем человеческое
- ❌ «Корпоративный enterprise-продукт» — доступен каждому, не только командам с бюджетом
- ❌ «Платформа» (по отношению к персональному IWE) — IWE = личная среда поверх общей платформы Aisystant

**Запрещённые слова для внешней коммуникации:**
Pack, DS, FPF, SPF, Цифровой двойник, ЦД, Платформа (про IWE), «связка с ИИ», «биологический интеллект». Полный список → `Уникальность IWE.md §«Запрещённые слова»`.

## 7. Смысловое ядро (нарратив)

> Развёрнутый нарратив для понимания — откуда выведены все формулировки. Источник: `Уникальность IWE.md §«Смысловое ядро»`. Здесь сводка ключевых утверждений.

**Главный переход эпохи ИИ:** навыки автоматизируются. Остаётся умение **конвертировать Проблему в Задачу через системное моделирование** (DP.D.053). ИИ-автопилот ускоряет «задача → работа», но не делает этот переход — выдаёт ответ на упрощённую интерпретацию. IWE — среда, где этот навык формируется в процессе реальной работы.

**Формула:** Созидательная мощь = человеческий интеллект × ИИ. ИИ — множитель. Мы растим первое.

**Три вещи в одном:**
1. Среда — платформа + личный IWE (помнит вас, растёт вместе с вами)
2. Методология — системное мышление, встроенное в инструмент
3. Экзоскелет — ИИ-агенты, которые усиливают мышление, не заменяют его

## 8. Открытые вопросы для валидации (Ф1 → Ф2)

1. **Archetype gate:** показать 5-10 людям из ЦА по три архетипа (Sage, Creator, Magician) без названий — попросить выбрать тот, который «звучит» как IWE. Подтверждает/опровергает Sage + Creator.
2. **Onliness test:** зачитать onliness statement холодному читателю. Вопрос: «Чем это отличается от Notion AI?» — если ответ требует объяснения больше 30 сек, переформулировать.
3. **JTBD verification:** в 5 глубинных интервью проверить формулировку «не помочь писать, а дать партнёра». Совпадает ли с реальным запросом?
