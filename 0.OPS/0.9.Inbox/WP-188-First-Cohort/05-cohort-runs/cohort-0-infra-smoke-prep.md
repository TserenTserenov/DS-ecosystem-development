## Участники

Дима, Ильшат, Наталья, Инга, Паша (ИТ-команда, совпадает с WP-253 Ф9).

## Критерий Г-П5 infra

- ≥4/5 дошли до Помощника без ручной поддержки менеджера >20 мин
- Время от входа в бот до первого ответа Помощника ≤60 мин (минимум), ≤30 мин (цель)

## Маршрут участника

1. Бот → клавиатура «Подключить Персону»
2. `/personal-guide-start` в Claude Code (или бот запускает wrapper) → создание flat-named `personal-guide-<username>` репо
3. Gateway OAuth: подключение через Claude-коннектор
4. Скилл заполняет 6 файлов из Персоны/Памяти (через 9 БД Neon по WP-253 Ф9)
5. Менеджер (Tseren) адаптирует руководство под каждого (ручная правка)
6. Диалог с Помощником (MCP-инструмент, личный контекст доступен)
