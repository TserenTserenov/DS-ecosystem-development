# Мировая платформа — принятые решения и план

> Документ для команды. Итог встречи 7 мая.

---

## Что решили

**Инфраструктура:** Google Cloud (GKE Standard, Европа). Не Autopilot — он дорогой при малом числе сервисов. Управление нодами вручную, пока нагрузка небольшая.

**База данных:** Cloud SQL (PostgreSQL). В том же регионе что и кластер — трафик между ними бесплатный.

**Сеть:** кластер работает в приватной сети, наружу торчит только один API Gateway с балансировщиком. Между Cloudflare и кластером — Origin-сертификаты Cloudflare.

**Деплой:** изучаем Werf. Разработчик пишет минимальный манифест (имя проекта + сервис), Werf сам собирает образ и деплоит в кластер.

**Инфраструктура как код:** Terraform Cloud, отдельный репозиторий.

**Репозитории:** отдельная организация в GitHub для мировой платформы. Не смешиваем с российскими.

**Аутентификация:** отдельный инстанс Ory для мировой платформы, не связан с российским. Пользователи существуют отдельно в каждой платформе.

**Данные:** никакой синхронизации между российской и мировой платформой. Если пользователь хочет перейти — экспортирует данные, импортирует в новую.

**Первые три месяца бесплатно:** Google даёт $300 кредита при первой регистрации.
